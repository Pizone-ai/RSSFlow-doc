üöÄ Installation Guide

Install from Browser Store

  1. Chrome Users: Visit Chrome Web Store
  2. Edge Users: Visit Microsoft Edge Add-ons
  3. Click "Add to Chrome" or "Get" button
  4. Confirm installation permissions
  5. After installation, the extension icon will appear in your browser toolbar

Initial Setup

  1. Click the extension icon to open the sidebar
  2. Click the settings button (gear icon)
  3. Select your preferred language
  4. Configure AI services (optional, for summary and chat features)
Tip: After first installation, we recommend adding a few RSS feeds to experience basic functionality before configuring AI features.

Quick Setup

  1. Go to Settings ‚Üí AI Settings ‚Üí Enable AI Summary
  2. Choose your AI provider (OpenAI, DeepSeek, Gemini, or SiliconFlow)
  3. Enter your API key
  4. Select a model (GPT-3.5+ recommended)
  5. Test the connection

  • DeepSeek: https://platform.deepseek.com/

  • SiliconFlow: https://account.siliconflow.cn/

‚ùì Frequently Asked Questions

For the complete list of 18 detailed FAQ items, please switch to the Chinese version. Here are the most important ones:

How to get activation codes for chat and podcast features?

+

Chat mode and podcast features offer a 5-day trial period. Activation codes are free and can be obtained by:

  • Leaving a positive review
  • Sharing or recommending the extension
  • Contacting the author directly

Important: Activation codes can only be used once and are bound to your browser. Don't delete the extension or browser after activation.

Why can't I see chat and podcast features?

+

These features require AI configuration and generated summaries. After configuring AI successfully, manually add an article to Marked to generate a summary, then the chat and podcast entries will appear in Flow view.

How to mark articles as read?

+

Double-click the left mouse button to mark as read. This is a core feature of RSSFlow and works in both title mode and immersive reading mode. You'll hear a wooden fish sound when marking as read.

Multi-language Support

RSSFlow supports 12 languages and can process content from any country in any language, returning analysis results in your selected interface language.

Data Analysis

You can analyze large amounts of data by adjusting the RSS content retention period and query date range in settings. Use data filtering by tags and keywords to optimize context and improve analysis accuracy.

üîß Troubleshooting

Common Issues

  • RSS feeds not updating: Check URL, network connection, and try refreshing
  • AI features not working: Verify API key, account balance, and model selection
  • Extension not loading: Refresh page, restart browser, or reinstall extension
  • Missing chat/podcast features: Ensure AI is configured and summaries are generated

Performance Tips

  • Regularly clean up old articles
  • Set reasonable retention periods
  • Limit number of RSS feeds
  • Use data filtering for better performance
