# 专家指令集 / Expert Command Set

### 中文

专家指令是**预调优的分析角色提示词**。在 AI 聊天或指令中心一键选用后，AI 会按该角色的视角、结构和关注点，对当前上下文中的 RSS 文章做深度加工。

与「高级功能 → 快捷指令」是同一能力：本页侧重**22 个内置角色说明**；快捷指令页侧重入口、自定义与模板变量。

> 部分高级指令需 **Pro** 激活后解锁。自定义指令可与系统指令合并出现在同一指令菜单中。

## 如何使用

1. 打开 **AI 聊天**（侧边栏、信息流分组、图谱等入口均可）。
2. 在输入区打开 **指令菜单 / 指令中心**。
3. 选中某一专家角色 → AI 立刻以该视角处理**当前加载的文章上下文**。
4. 需要固定工作流时，可在 **设置** 中新建自定义指令，并与内置指令一起显示。

## 自定义指令

- 在设置中的指令管理里 **新增 / 编辑 / 删除** 个人指令。  
- 自定义项会与 22 条系统指令 **合并显示** 在同一菜单。  
- 模板可使用变量（详见 **快捷指令**）：`{{summary_data}}`、`{{original_context}}`、`{{history}}` 等。

## 使用提示

- 指令效果依赖上下文：优先在 **信息流分组聊天** 或已加载相关文章的会话中使用。  
- 先开 **自动摘要**，再跑专家指令，通常比直接塞全文更稳、更省 Token。  
- 需要可引用结论时，配合聊天中的 **引文徽标** 回看原文。

### English

Expert Commands are **pre-tuned analysis roles**. Pick one in AI Chat / Command Center and the model rewrites its lens, structure, and priorities over the articles currently in context.

Same capability as **Advanced → AI Commands**: this page is the **catalog of 22 built-ins**; that page covers entry points, custom prompts, and template variables.

> Some advanced commands require **Pro**. Custom prompts merge into the same menu.

## How to use

1. Open **AI Chat** (sidebar, Flow group, Graph, etc.).
2. Open the **command menu / Command Center**.
3. Select a role → the model analyzes the **current article context** in that style.
4. Add personal prompts under **Settings**; they appear next to built-ins.

## Built-in commands (22)

### Creation & distribution

| Command | What it does | Best for | Typical output |
|---------|--------------|----------|----------------|
| **Ideator** | Clusters multi-source items into expandable topic maps and angles | Content planning, editorial calendars | Topic clusters, angle list |
| **Trend Selector** | Ranks what is timely and high-spread potential | Daily hot-topic triage | Priority ranking, brief rationale |
| **Draft Builder** | Turns selected materials into a structured outline or first draft | Long-form starts, weekly writeups | Outline + draft body |
| **Humanizer** | Reduces robotic tone; improves flow and natural voice | Pre-publish polish | Readable rewrite |
| **Threader** | Compresses research into X/Twitter-style threads | Social distribution | Numbered thread + hooks |

### Research & facts

| Command | What it does | Best for | Typical output |
|---------|--------------|----------|----------------|
| **Fact Brief** | Cross-checks conflicting reports; separates verified vs unresolved | Contested events | Verified / disputed / unknown |
| **De-noiser** | First-principles extraction of invariants from noisy long text | Papers, PR-heavy posts | Core claims + key numbers |
| **Dynamics** | Causal chains and feedback loops across variables | Policy–market linkage | Causal narrative / loop notes |
| **Domino Sim** | Multi-hop supply-chain / industry shock propagation | Black-swan spillover | 1st / 2nd / 3rd order impacts |

## Custom commands

Create/edit/delete under Settings. Customs merge with the 22 system roles in one menu. Template variables are documented under **AI Commands** (`{{summary_data}}`, `{{original_context}}`, `{{history}}`).

## Tips

- Load a **relevant article set** (e.g. Flow group chat) before running a command.  
- **Auto-summary first** usually beats dumping raw full text (quality + tokens).  
- Use **citation badges** in chat when you need to verify claims against sources.
