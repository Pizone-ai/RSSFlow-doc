# 快捷指令 / AI Commands

### 中文

快捷指令（指令中心）用于对**单篇或多篇文章**一键调用预置 / 自定义提示词，完成提炼、改写或专业研判。

**22 条内置专家角色的详细说明**（做什么、适用场景、典型产出）见：

> **AI 智能核心 → 专家指令集**

本页说明：**入口、自定义、模板变量**。

## 入口

- AI 聊天 / 阅读侧栏底部的 **指令中心**  
- **设置** 中的指令管理（创建、编辑、删除）  
- 信息流分组、图谱等对话上下文中同样可打开指令菜单

## 自定义指令

可按个人工作流新增指令。

| 字段 | 说明 |
|------|------|
| **指令名称** | 菜单中显示的短名称 |
| **提示词模板** | 发给模型的完整要求（System / 主提示） |
| **上下文开关** | 是否自动附带摘要、对话历史等 |

### 模板变量

运行时会替换为真实内容：

| 变量 | 含义 |
|------|------|
| `{{summary_data}}` | 当前文章的摘要与基础元数据 |
| `{{original_context}}` | 原文全文（按实现可能截断） |
| `{{history}}` | 链式 / 多步任务中，上一步 AI 输出 |

> 编写自定义指令时写清：**角色、输入、输出结构、禁止事项**，效果通常更稳定。

## 使用建议

1. 先保证相关文章已进入当前对话上下文。  
2. 优先依赖已生成的 **AI 摘要**，再跑专家指令。  
3. 需要可核验结论时，使用聊天中的 **引文** 回看原文。  

### English

AI Commands run preset or custom prompts over one or many articles.  
**Full role catalog (what / when / output)** lives in **AI Smart Core → Expert Command Set**. This page covers access, custom prompts, and variables.

## Access

Command Center in chat/reader; manage library under Settings; also available in Flow/Graph chat contexts.

## Built-in quick list

22 roles (Ideator, Trend Selector, Draft Builder, Humanizer, Risk/Sector/Macro radars, Forex Quant, CSO Board, Crypto Intel, Fact Brief, Dynamics, Threader, Market Brief, Tech Daily, De-noiser, Tech Onboarding, Mind Coach, Domino Sim, Lindy Filter, Serenity Method, Bayesian Trade). Some require **Pro**.

## Custom commands

| Field | Purpose |
|-------|---------|
| Name | Menu label |
| Prompt template | Full instruction to the model |
| Context toggles | Attach summary / chat history |

### Variables

| Variable | Meaning |
|----------|---------|
| `{{summary_data}}` | Summary + metadata |
| `{{original_context}}` | Source text (may be truncated) |
| `{{history}}` | Previous step output in chains |

## Tips

Load relevant context first; prefer auto-summaries before heavy commands; use citation badges to verify claims.
