# 跨扩展同步 / Cross-Extension Sync

### 中文

把已处理的文章元数据与 AI 结果，推送给本机其他已授权浏览器扩展，实现工具链联动。

## 入口

**设置 → 跨扩展同步**

## 选项

| 项 | 说明 |
|----|------|
| 启用同步 | 总开关 |
| 目标扩展 ID | 白名单；填 `*` 表示允许本机扩展查询（需谨慎） |
| 自动推送 | 新摘要完成后自动广播 |

## 共享内容

- 基础：标题、链接、日期、订阅源  
- AI：优化标题、摘要、要点、观点  
- 标签与评分  

## 步骤

1. 在 `chrome://extensions`（或 Edge 对应页）复制目标扩展 32 位 ID  
2. 粘贴到 RSSFlow 目标 ID  
3. 打开 **启用同步** 与 **自动推送**  

### English

Stream processed articles and AI fields to other **authorized** extensions on the same browser.

## Access

**Settings → Sync**

## Options

Enable sync, whitelist target extension ID (`*` allows broader local queries—use carefully), and auto-push after summaries.

## Payload

Metadata (title, URL, date, feed) plus AI summary, bullets, opinions, tags, and score.

## Setup

Copy the 32-char ID from the extensions page → paste in RSSFlow → enable sync and auto-push.
