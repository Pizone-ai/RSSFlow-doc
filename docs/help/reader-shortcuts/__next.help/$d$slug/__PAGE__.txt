<h2>输入焦点保护</h2>
<p>以下情况不触发快捷键：</p>
<ul>
<li>按住 <code>Ctrl</code> / <code>Cmd</code> / <code>Alt</code>（保留浏览器快捷键，如刷新、查找）</li>
<li>焦点在输入框、文本域、下拉框中</li>
<li>焦点在可编辑区域（contentEditable）</li>
</ul>
<p>导航类按键（<code>J</code> / <code>K</code> / <code>R</code> / 方向键）会忽略长按连发。</p>
