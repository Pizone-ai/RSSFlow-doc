<li><strong>DeepSeek</strong> — 高性价比推理与聊天</li>
<li><strong>OpenAI</strong> — GPT 系列</li>
<li><strong>Anthropic Claude</strong> — 长文逻辑与研报风格</li>
<li><strong>Google Gemini</strong> — 大上下文</li>
<li><strong>OpenAI Compatible</strong> — 兼容中转 / 聚合（如 SiliconFlow、火山引擎）</li>
<li><strong>Local LLM</strong> — Ollama / LM Studio 等本机模型</li>
</ul>
