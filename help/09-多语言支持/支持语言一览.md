# 支持语言一览 / Supported Languages

### 中文

界面与 AI 输出可切换多种语言（随版本可能增减）。

## 常见语言

| 语言 | 代码 |
|------|------|
| English | `en` |
| 简体中文 | `zh-cn` |
| 繁體中文 | `zh-tw` |
| 日本語 | `ja` |
| 한국어 | `ko` |
| Español | `es` |
| Português | `pt-br` |
| Français | `fr` |
| Deutsch | `de` |
| Italiano | `it` |
| Русский | `ru` |
| العربية | `ar` |
| हिन्दी | `hi` |
| Nederlands | `nl` |

## 如何切换

**设置 → 常规 → 语言**

## 影响范围

- 侧边栏、选项页、聊天等 UI 文案  
- AI 摘要、对话、定时任务报告、热点发现的语言偏好  

## 首次启动

默认跟随浏览器语言；手动选择后覆盖自动检测，并保存在本地。

### English

Switch UI and AI output language under **Settings → General → Language**.

Common codes: `en`, `zh-cn`, `zh-tw`, `ja`, `ko`, `es`, `pt-br`, `fr`, `de`, `it`, `ru`, `ar`, `hi`, `nl` (availability may vary by version).

Affects UI strings plus AI summaries, chat, workflow reports, and discovery copy. First launch follows the browser locale until you override it.
