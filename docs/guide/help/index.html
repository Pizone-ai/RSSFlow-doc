ğŸš€ Installation Guide

Install from Browser Store

  1. Chrome Users: Visit Chrome Web Store
  2. Edge Users: Visit Microsoft Edge Add-ons
  3. Click "Add to Chrome" or "Get" button
  4. Confirm installation permissions
  5. After installation, the extension icon will appear in your browser toolbar

Initial Setup

  1. Click the extension icon to open the sidebar
  2. Click the settings button (gear icon)
  3. Select your preferred language
  4. Configure AI services (optional, for summary and chat features)
Tip: After first installation, we recommend adding a few RSS feeds to experience basic functionality before configuring AI features.

ğŸ“– Basic Usage

Adding RSS Feeds

  1. Click the "+" button at the top of the sidebar
  2. Enter the RSS feed URL
  3. Click the "Add" button
  4. The system will automatically fetch and display the latest articles

Reading Articles

  • Mark as Read: Double-click article title or press M key
  • Bookmark Article: Click star icon or press S key
  • Open Original: Click article title or press O key
  • Expand Full Text: Click expand button or press E key

Keyboard Shortcuts

J - Next article K - Previous article M - Mark as read/unread S - Bookmark/unbookmark O - Open original link E - Expand/collapse full text R - Refresh feeds F - Search 1 - All articles view 2 - Bookmarked view 3 - DayFlow view 4 - TagFlow view

View Modes

  • All View: Shows all unread articles
  • DayFlow: Smart grouping by date
  • TagFlow: AI-based tag classification
  • Marked: Shows bookmarked articles

ğŸ¤– AI Features Configuration

Supported AI Models

  • OpenAI: GPT-3.5, GPT-4, etc.
  • DeepSeek: Cost-effective Chinese-optimized models
  • Google Gemini: Google's multimodal AI models
  • SiliconFlow: Domestic AI service platform
  • Others: All models supporting OpenAI API format

Configure AI Services

  1. Open settings page (click gear icon)
  2. Select "AI Configuration" tab
  3. Choose your AI service provider
  4. Enter API key
  5. Select model (recommend GPT-3.5+ level models)
  6. Click "Test Connection" to verify configuration

Using AI Features

Auto Summary

  • Enable "Auto Summary" in RSS feed settings
  • New articles will automatically generate AI summaries
  • Support custom summary prompts

Chat Dialogue

  • Select article and click chat icon
  • Ask questions about the article
  • Support multi-turn conversations
  • Provide quick command templates

Voice Podcast

  • Select articles or date range
  • Click podcast generation button
  • AI generates podcast script and reads aloud
  • Support multiple voice styles
Note: AI features consume API calls. Please configure according to your usage needs. We recommend using models with GPT-3.5+ capabilities for stable output.

ğŸ”§ Troubleshooting

Common Issues

Cannot add RSS feeds or update failures

  • Check if RSS feed URL is correct
  • Confirm website supports RSS/Atom format
  • Try refreshing or re-adding
  • Check network connection

AI features not working

  • Check if API key is correctly configured
  • Confirm API account has sufficient balance
  • Check network connection
  • Try switching to other AI models

Extension fails to load properly

  • Refresh browser page
  • Restart browser
  • Check if extension is disabled
  • Try reinstalling the extension

Data loss or sync issues

  • Data is stored locally, no automatic sync
  • Export data backup in settings
  • Backup data before reinstalling extension

Performance Optimization Tips

  • Regularly clean up expired articles
  • Set reasonable article retention period
  • Avoid adding too many RSS feeds
  • Turn off unnecessary AI auto-summaries
