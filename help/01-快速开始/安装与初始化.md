# 安装与初始化

## 安装方式

RSSFlow Pro **仅支持**从官方商店安装，不支持开发者模式加载或自行构建：

- **Chrome 网上应用店**：
  `https://chromewebstore.google.com/detail/rssflow-ai-powered-rss-in/mefbfkpippglgoanjcbdjnkelcbdjija`
- **Edge 外接程序**：
  `https://microsoftedge.microsoft.com/addons/detail/rssflow-aipowered-rss-/khgllclaeabkjgoblcipfpgaejblcelf`

## 首次使用

1. 安装完成后，点击浏览器工具栏的扩展，找到rssflow，点击固定到工具栏，方便后续点击浏览器工具栏中的 RSSFlow 图标 和查看未读文章数角标
2. 点击扩展图标后，侧边栏会自动打开，进入到rssflow的主界面
3. **建议**：在浏览器工具栏中将 RSSFlow 固定（pin），以便快速访问

## 注意

- 侧边栏是扩展的主界面，非弹出式窗口（popup）
- 再次点击工具栏图标可关闭/打开侧边栏

# Installation & Initialization

## Installation

RSSFlow Pro is **only** available through official stores. Developer mode or self-building is not supported:

- **Chrome Web Store**:
  `https://chromewebstore.google.com/detail/rssflow-ai-powered-rss-in/mefbfkpippglgoanjcbdjnkelcbdjija`
- **Edge Add-ons**:
  `https://microsoftedge.microsoft.com/addons/detail/rssflow-aipowered-rss-/khgllclaeabkjgoblcipfpgaejblcelf`

## First Use

1. After installation, click the RSSFlow icon in the browser toolbar
2. The sidebar opens automatically (via Chrome `sidePanel` API with `setPanelBehavior({ openPanelOnActionClick: true })`)
3. **Recommended**: Pin RSSFlow to the toolbar for quick access

## Notes

- The sidebar is the main extension interface, not a popup
- Click the toolbar icon again to close/reopen the sidebar
