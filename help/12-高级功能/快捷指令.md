# 快捷指令 / AI Commands

### 中文

快捷指令（AI 助手指令）是 RSSFlow Pro 的核心提效工具。系统内置了 22 个针对不同行业深度分析的智能专家指令，并且允许您自定义专属指令，一键对单篇文章或多篇文章组合进行深度提炼、格式改写或逻辑研判。

**使用入口：** 
- 在阅读文章或进行 AI 聊天时，点击右侧面板底部的“指令中心”。
- 您也可以在设置页面中，进入“快捷指令”管理面板进行创建与编辑。

**系统内置的 22 个专家级指令一览：**

| 指令名称 | 英文名 | 核心用途与场景 |
|----------|-------|----------------|
| 价值选题 | Ideator | 多源聚合，帮助自媒体与创作者规划长线价值选题。 |
| 热点选题 | Trend Selector | 自动筛选社交网络及各渠道的当日高光热点。 |
| 写文章 | Draft Builder | 针对选中的多篇素材，自动合成一篇深度整合文章首稿。 |
| 去 AI 腔 | Humanizer | 抹除大模型机械化语调，进行拟人化口语重写与排版优化。 |
| 系统风险 | Risk Radar | 评估地缘政治、合规性及业务链条的潜在系统性风险。 |
| 行业透视 | Sector Radar | 跨板块进行竞争格局、市场占有率及产品壁垒分析。 |
| 宏观透视 | Macro View | 解析各国央行货币政策、宏观经济走向与通胀环境。 |
| 外汇研判 | Forex Quant | 监测全球汇率变动、利差走势与货币溢价。 |
| 战略决策 | CSO Board | 深度分析企业商业模式，审计企业护城河与竞争优势。 |
| 加密投研 | Crypto Intel | 分析区块链链上筹码分布、DeFi 项目逻辑与代币模型。 |
| 证据审查 | Fact Brief | 针对矛盾冲突的信息源，进行事实证据交叉印证与排噪。 |
| 传导推演 | Dynamics | 进行因果推演，帮助梳理事件的连锁反应与反馈环路。 |
| 写推特 | Threader | 将长文提炼为适合社交平台发布的系列短帖文（Thread）。 |
| 市场研报 | Market Brief | 梳理行业上下游因果传导图，输出结构化市场研报。 |
| 科技日报 | Tech Daily | 跟踪科技巨头动向，深度审计平台经济体与数字化变革。 |
| 学术论文 | De-noiser | 过滤学术论文中的冗长陈述，精准提取核心公式、参数与结论。 |
| 技术博主 | Tech Onboarding | 针对新技术文档或开源框架，快速评估上手成本与迁移摩擦。 |
| 心智教练 | Mind Coach | 提供行动指南，帮助诊断并改善阅读拖延与心智内耗。 |

**自定义指令 (Custom Commands)：**

您可以随时根据个人工作流创建个性化的指令。

**编辑字段说明：**
- **指令名称**：易于识别的快捷名称。
- **提示词模板**：在此输入您对 AI 的详细要求（即 System Prompt）。在模板中，您可以使用花括号插入动态变量，系统在运行时会自动替换为实际内容：
  - `{{summary_data}}`：自动替换为当前文章的详细摘要与基础信息。
  - `{{original_context}}`：自动替换为文章的原文全文。
  - `{{history}}`：在链式（多步）处理中，自动替换为前一个步骤 of AI 输出内容。
- **包含摘要/包含对话历史**：勾选后，系统会自动把上下文和对话历史作为背景发送给大模型，避免丢失上下文。

### English

AI Commands are pre-configured, context-aware prompt templates in RSSFlow Pro. The extension features 22 professional-grade preset workflows for research, finance, writing, and risk analysis, while allowing you to build and chain your own custom instructions.

**Access:**
- Click "Command Center" at the bottom of the AI Chat panel.
- Go to the Settings page → "AI Commands" to manage your library.

**The 22 Built-in Expert Commands:**

| Command | Usage & Scenarios |
|---------|-------------------|
| **Ideator** | Aggregates insights from multiple sources to plan long-term high-value content. |
| **Trend Selector** | Curates and extracts trending topics from social networks and feed streams. |
| **Draft Builder** | Synthesizes complex articles and research briefs into structured first drafts. |
| **Humanizer** | Polishes machine-sounding AI outputs into natural, engaging, and well-formatted text. |
| **Risk Radar** | Evaluates geopolitical, regulatory, and systemic compliance risks for markets. |
| **Sector Radar** | Compares competitive landscapes, market shares, and product moats. |
| **Macro View** | Analyzes central bank policies, macroeconomic indicators, and inflation trends. |
| **Forex Quant** | Monitors global exchange rates, interest rate spreads, and currency valuations. |
| **CSO Board** | Audits business models, defensive moats, and corporate strategic directions. |
| **Crypto Intel** | Examines on-chain metrics, smart money trends, DeFi protocols, and token economics. |
| **Fact Brief** | Cross-references conflicting news reports to verify facts and filter out noise. |
| **Dynamics** | Maps out feedback loops and models causal networks of political/economic events. |
| **Threader** | Distills long-form reading material into ready-to-share social media threads. |
| **Market Brief** | Creates causal supply-chain maps and outputs structured industry research summaries. |
| **Tech Daily** | Tracks tech giants, platform economics, and software business model evolution. |
| **De-noiser** | Strips academic papers of fluff, extracting formulas, parameters, and key evidence. |
| **Tech Onboarding** | Evaluates migration friction, learning curves, and adoption costs for new tools. |
| **Mind Coach** | Diagnoses reading procrastination and provides actionable cognitive exercises. |
| **Domino Sim** | Simulates supply-chain shocks and domino-effect disruptions. |
| **Lindy Filter** | Filters out passing trends to focus on high-durability, timeless knowledge. |
| **Serenity Method** | Audits manufacturing supply-chains, BOM costs, and technical bottlenecks. |
| **Bayesian Trade** | Guides decision-making and risk hedging based on probabilistic models. |

**Custom Commands:**

You can create personalized commands tailored to your specific analysis tasks.

**Configuration Fields:**
- **Name**: The display name for your shortcut.
- **Prompt Template**: The core instruction sent to the AI. You can insert dynamic variables wrapped in double curly braces:
  - `{{summary_data}}`: Replaced with the current article's generated summary and metadata.
  - `{{original_context}}`: Replaced with the full text of the source article.
  - `{{history}}`: In chained multi-step tasks, replaced with the output of the previous processing step.
- **Context Toggles**: Options to automatically append the original article summary or active chat history as background context for the AI.
