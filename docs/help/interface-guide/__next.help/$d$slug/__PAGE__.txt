<p>RSSFlow Pro 以浏览器 <strong>侧边栏</strong> 形式运行，包含四个标签页与底部工具栏。</p>
<h2>四个标签页</h2>
<table><thead><tr>
<th>标签</th>
<th>说明</th>

</tr>
<tr>
<td>收藏</td>
<td>已收藏的文章</td>
</tr>
<tr>
<td>图谱</td>
<td><strong>在新标签页</strong>打开知识图谱，便于多关键词组合搜索</td>
</tr>
<tr>
<td>Flow 视图</td>
<td><strong>在新标签页</strong>按日期 / 标签查看 AI 处理后的信息流</td>
</tr>
</tbody></table>
<blockquote><p>点击「图谱」或「Flow 视图」时，会在新浏览器标签中打开，而不是在侧边栏内嵌显示。</p></blockquote>

</tr>

</tr>

</tr>
<tr>
<td>AI 聊天</td>
<td>进入对话；可切换顶部分组，或在左侧过滤数据</td>
</tr>
<tr>
<td>设置</td>
<td>打开选项页</td>
</tr>
<tr>
<td>主题</td>
<td>浅色 / 深色 / 跟随系统</td>
</tr>
</tbody></table>
