# 定时任务概述 / Workflow Overview

### 中文

在后台按计划抓取、用 AI 分析，并把简报推送到手机或群聊。

## 入口

**设置 → 工作流**

## 界面

- **任务列表**：启用 / 禁用、编辑、删除；可手动立即运行  
- **执行历史**：最近约 50 条运行日志  

## 核心机制

| 点 | 说明 |
|----|------|
| 排队执行 | 多任务串行，避免同时打爆 API |
| 调度 | 固定间隔（如每 12 小时）或指定时间 |
| 推送 | 每任务可配 Telegram / 飞书 / 桌面通知 / Webhook |
| 额度 | 免费约 2 个活跃任务；专业版更高 |

相关：**三种执行模式**、**三种报告发布模式**、**标签过滤与内容筛选**。

### English

Schedule background fetch + AI analysis and push briefings to your channels.

## Access

**Settings → Workflow**

## UI

- **Workflows**: enable/disable, edit, delete, or run now  
- **Journal**: last ~50 execution logs  

## Core Behavior

| Topic | Detail |
|-------|--------|
| Queue | Serial execution to avoid API rate limits |
| Schedule | Intervals or specific times |
| Push | Telegram, Feishu, desktop notifications, webhooks |
| Limits | ~2 active workflows on free tier; more on Pro |

See also: Execution Modes, Report Publish Modes, Filters & Content Selection.
