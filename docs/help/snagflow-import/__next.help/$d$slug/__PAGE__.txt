2:T674,<h2>入口</h2>
<p><strong>设置 → 订阅源 →「从 SnagFlow 订阅生成器导入」</strong>（可折叠面板）</p>
<h2>前置条件</h2>
<p>已安装 <a href="https://chromewebstore.google.com/detail/snagflow-rss-converter/beafcjcdnfeajjpmienpdjmeabhjnljh" rel="noopener noreferrer" target="_blank">SnagFlow</a> 扩展。</p>
<h2>操作步骤</h2>
<ol>
<li><strong>展开面板</strong>：点击「从 SnagFlow 订阅生成器导入」。</li>
<li><strong>确认扩展 ID</strong>：默认已预填官方扩展 ID；若使用其他实例可修改。</li>
<li><strong>加载规则</strong>：点击 <strong>加载规则</strong>。RSSFlow 与 SnagFlow 通过浏览器扩展间通信读取规则，<strong>不依赖外网接口</strong>。</li>
<li><strong>选择规则</strong>：列表中每项包含标题、描述、订阅 URL 与状态：</li>
</ol>
<ul>
<li><code>已订阅</code>：已在 RSSFlow 中，不可再选</li>
<li><code>已禁用</code>：SnagFlow 侧已关闭，不可选</li>
</ul>
<ol>
<li><strong>添加选中</strong>：勾选后点击 <strong>添加选中的订阅源</strong>。</li>
</ol>
<h2>附加说明</h2>
<ul>
<li>面板头部实时显示已选数量</li>
<li>已订阅源自动禁用勾选</li>
<li>面板支持展开 / 收起</li>
</ul>
