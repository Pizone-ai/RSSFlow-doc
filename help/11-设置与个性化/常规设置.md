# 常规设置 / General Settings

### 中文

控制抓取频率、列表时间范围、历史保留、界面语言与音效。

## 入口

**设置 → 常规**

## 配置项

| 项 | 范围 / 说明 | 建议 |
|----|-------------|------|
| **更新间隔** | 1–1440 分钟，后台拉取所有订阅源 | 新闻源可 60 分钟；省资源可 240 分钟+ |
| **查询范围** | 1–365 天，侧边栏 / Flow 默认加载窗口 | 源很多时用 7–14 天提升流畅度 |
| **最大保存时间** | 1–365 天，超期文章自动清理 | 常用 30–90 天；磁盘紧张则缩短 |
| **界面语言** | 手动选择；默认跟随浏览器 | 也会影响 AI 摘要 / 报告语言偏好 |
| **音效** | 双击已读时的木鱼提示音 | 可按喜好开关 |

### English

Controls fetch interval, list window, retention, UI language, and sound.

## Access

**Settings → General**

## Options

| Setting | Range / role | Tips |
|---------|--------------|------|
| **Update interval** | 1–1440 minutes | ~60m for news; longer to save resources |
| **Query range** | 1–365 days loaded in lists / Flow | 7–14 days if you have many feeds |
| **Retention** | 1–365 days before auto-purge | 30–90 days typical |
| **Language** | UI language (default: browser) | Also steers AI summary language |
| **Sound** | Wooden-bell on double-click dismiss | Toggle as preferred |
