# 高级参数说明 / Generation Parameters

### 中文

在大模型设置中，如果您希望精细调节 AI 回答的风格、创造力以及篇幅长度，可以点击 **“显示生成参数”** 展开高级参数调节面板。

**核心生成参数说明：**
系统提供以下三个参数滑块供您精细微调：

- **多样性 (Temperature / 温度值)**：
  - **范围**：0.0 - 2.0（默认值 0.7）
  - **说明**：控制 AI 输出内容的创造力与随机性。较低的数值（如 0.3）会让回答更加严谨、固定和切题；较高的数值（如 0.9）会让回答更加天马行空、充满创意和随机变化。
  - **建议**：文章摘要和事实提炼建议设为 **0.3 - 0.5**；创意写作和观点脑暴建议设为 **0.7 - 0.9**；日常对话建议使用默认的 **0.7**。
- **核采样 (Top P)**：
  - **范围**：0.0 - 1.0（默认值 0.7）
  - **说明**：另一种控制回答确定性的方式。控制 AI 在选择词汇时只考虑累计概率达到该阈值的词。一般情况下，您**只需要调节多样性 (Temperature) 即可**，保持 Top P 为默认值 0.7。
- **单次生成长度上限 (Max Tokens)**：
  - **范围**：100 - 65,536（默认值：基础模型 8,192 / 复杂模型 20,000）
  - **说明**：限定 AI 单次输出的字数长度。如果设置得过小，AI 的回答可能在没有写完时被强行截断；如果设置得过大，则会影响后台并发处理的速度。建议使用系统为您推荐的默认预设。

**恢复默认值：**
在标签设置栏的右侧，点击“恢复默认设置”可以一键将所有的系统标签范围还原为出厂预设值。

### English

In AI Settings, you can click **"Show Generation Parameters"** to reveal fine-tuning controls. These parameters shape the personality, creativity, and length of AI-generated responses.

**Key Generation Parameters:**
You can customize the following three sliders:

- **Creativity (Temperature)**:
  - **Range**: 0.0 - 2.0 (Default: 0.7)
  - **Overview**: Controls how creative or deterministic the AI's language is. Lower values (e.g., 0.3) make answers rigid, factual, and tightly focused. Higher values (e.g., 0.9) increase creativity, flavor, and randomness.
  - **Guidelines**: Use **0.3 - 0.5** for summaries and fact extraction; use **0.7 - 0.9** for creative writing and brainstorming; use **0.7** for general chat.
- **Nucleus Sampling (Top P)**:
  - **Range**: 0.0 - 1.0 (Default: 0.7)
  - **Overview**: An alternative way to guide output diversity. We recommend keeping Top P at its default value and **adjusting only Temperature** to customize creativity.
- **Max Generation Length (Max Tokens)**:
  - **Range**: 100 - 65,536 (Default: 8,192 for Base, 20,000 for Advanced)
  - **Overview**: Limits the maximum length of a single AI response. If set too low, the AI's response might get cut off mid-sentence. If set too high, it might slow down concurrent processing speeds. The default presets are highly recommended.

**Restoring Defaults:**
Clicking "Restore Defaults" next to the tag scope section instantly resets the system classification tags to their original out-of-the-box values.
