# 知识图谱模式 / Graph View

### 中文

用标签与共现关系把文章连成可视化网络，发现跨主题交集。

## 入口

侧边栏 **图谱**（新标签打开）；工具栏也可进入。

## 搜索

支持逻辑组合：

- 空格 / `AND`：同时包含  
- `OR`：任一  
- `NOT`：排除  

`Alt+K` 聚焦搜索框；`Esc` 退出搜索。结果会实时重组节点。

## 图谱阅读

| 元素 | 含义 |
|------|------|
| 气泡节点 | 主题标签 |
| 连线粗细 | 共现频率 |
| 琥珀色高亮 | 跨领域桥接概念 |
| 颜色 | 目录词 / 热词 / 当前焦点等 |

可拖拽、缩放；点击节点查看关联文章。

## 时间线

选中节点或边后，右侧按时间列出文章；滑块筛选时段，可播放演进。

## 图谱对话

对当前检索结果发起 AI 聊天；提供建议问题；回答支持引文预览与跳转。

### English

Visual association network of tags and co-occurrence across RSS articles.

## Access

Sidebar **Graph** (new tab) or toolbar entry.

## Search

`AND` / `OR` / `NOT` logic; `Alt+K` focuses search; graph re-clusters live.

## Reading the Web

Nodes = tags; edge thickness = co-occurrence; amber = cross-domain bridges. Pan/zoom; click for related articles.

## Timeline & Chat

Filter by time slider; discuss the current selection with cited AI chat.
