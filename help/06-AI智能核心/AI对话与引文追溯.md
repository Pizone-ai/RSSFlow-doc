# AI 对话与引文追溯 / AI Chat & Citation Tracing

### 中文

基于选定的 RSS 文章集合与 AI 对话，并在回答中通过引文徽标追溯原文。

## 如何打开

| 入口 | 说明 |
|------|------|
| 侧边栏 **AI 聊天** | 在新标签打开对话 |
| 信息流分组右下角 **AI 聊天** | 以该日期 / 标签组文章为上下文 |
| 知识图谱页右侧 | 在图谱结果上继续对话 |
| 热点 / 发现面板 | 将分析报告带入聊天继续追问 |

## 引文追溯

- AI 引用本地文章时，正文会出现 `[1]`、`[2]` 等徽标  
- **悬停**：查看标题与摘要  
- **点击**：新标签打开原文核对  

## 能力要点

- **流式输出**：边生成边渲染 Markdown  
- **上下文压缩**：文章过多时自动压缩核心片段，控制长度与费用  
- **会话历史**：本地保存，可继续上次对话  
- **专家快捷指令**：内置多套专业指令（部分需专业版）  

### English

Chat over selected RSS articles and trace claims back to sources via citation badges.

## Access Points

| Entry | Behavior |
|-------|----------|
| Sidebar **AI Chat** | Opens chat in a new tab |
| Flow group **Chat** | Uses that date/tag group as context |
| Graph page sidebar | Chat over graph results |
| Discovery panel | Continue from an intelligence report |

## Citation Tracing

- Badges like `[1]`, `[2]` mark cited articles  
- **Hover** for title + summary  
- **Click** to open the original article  

## Features

- Streaming Markdown responses  
- Smart context compression for large sets  
- Local session history  
- Expert command palette (some commands require Pro)
