# 三种执行模式详解 / Execution Modes

### 中文

在工作流编辑器中可选择三种执行模式，按需聚合与加工文章。

## 1. 单指令分析 (Single)

- **做法**：选一条 AI 指令（如「每日简报」），多篇文章一次汇总成一份报告  
- **适合**：日常要闻、轻量简报  

## 2. 串行链式 (Sequential Chain)

- **做法**：步骤一 → 步骤二…，上一步输出作为下一步输入  
- **适合**：多阶段深加工（先提要点，再写综述 / 翻译 / 扩写）  

## 3. 并行汇总 (Split-Merge)

- **分析阶段**：多维度并行分析同一批文章（如技术影响 + 市场机会）  
- **汇总阶段**：合并各维度结果，形成综合决策报告  
- **适合**：多视角研报与交叉分析  

### English

Three execution modes in the workflow editor.

## 1. Single

One command aggregates many articles into a single brief. Best for daily roundups.

## 2. Sequential Chain

Pipeline steps where each step consumes the previous output. Best for multi-stage polish (extract → rewrite / translate).

## 3. Split-Merge

Parallel dimensional analyses (map), then a final synthesis (reduce). Best for multi-perspective research.
