# 卡片阅读与多列自适应 / Card Layout & Multi-column

### 中文

侧边栏与信息流中的文章以 **卡片** 展示。可根据侧边栏宽度自动分列，并在标准 / 极简两种密度间切换。

## 标准模式卡片

典型结构（自上而下）：

1. **首行**：订阅源名称 · AI 指示（有摘要 / 无摘要）· 相对时间 · 收藏按钮（悬停显示）
2. **标题**：醒目标题，悬停高亮
3. **摘要**：优先显示 AI 摘要，否则显示原文摘要，最多约 3 行
4. **缩略图**（可选）：右侧小图，悬停可预览大图

极简模式仅保留 AI 指示点、标题与时间，适合快速扫读。

## 多列自适应

侧边栏变宽时自动增加列数（约每 340px 一列）：

- 约 ≥ 680px：2 列
- 约 ≥ 1020px：3 列

列表使用虚拟滚动，大量文章时仍保持流畅。  
拖拽调整侧边栏宽度时，列数会暂时冻结，松手后再重新计算。

## 标准 vs 极简

| 特性 | 标准 | 极简 |
|------|------|------|
| 卡片高度 | 完整（摘要 + 可选缩略图） | 紧凑（标题行） |
| 摘要 | 约 3 行截断 | 不显示 |
| 缩略图 | 显示 | 不显示 |
| AI 指示 | 图标 | 色点 |
| 收藏 | 悬停显示 | 不显示 |
| 日期分组 | 完整分组样式 | 简约标签 |

通过工具栏 **视图模式** 切换；选择会保存在本地。

### English

Articles appear as **cards** in the sidebar and Flow. Column count adapts to sidebar width; density switches between Standard and Mini.

## Standard Card

Top to bottom:

1. **Header**: feed name · AI indicator · relative time · favorite (on hover)
2. **Title**: emphasized, highlight on hover
3. **Summary**: AI summary preferred, else snippet (~3 lines)
4. **Thumbnail** (optional): small image with hover preview

Mini mode keeps only the AI dot, title, and time for fast scanning.

## Multi-column Grid

Wider sidebars add columns (~one column per 340px):

- ~≥ 680px: 2 columns
- ~≥ 1020px: 3 columns

Virtualized lists keep long feeds smooth. Column count freezes while you resize, then recalculates on release.

## Standard vs Mini

| Feature | Standard | Mini |
|---------|----------|------|
| Height | Full (summary + optional image) | Compact title row |
| Summary | ~3-line clamp | Hidden |
| Thumbnail | Shown | Hidden |
| AI indicator | Icon | Dot |
| Favorite | Hover | Hidden |
| Date groups | Full style | Compact labels |

Toggle via the toolbar **view mode** control; the choice is stored locally.
