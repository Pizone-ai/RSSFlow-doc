<h2>配置步骤</h2>
<h3>1. 创建机器人</h3>
<ol>
<li>在 Telegram 搜索 <strong>@BotFather</strong></li>
<li>发送 <code>/newbot</code>，按提示设置名称与用户名（用户名以 <code>bot</code> 结尾）</li>
<li>复制返回的 <strong>Bot Token</strong></li>
</ol>
<h3>2. 获取 Chat ID</h3>
<ul>
<li><strong>个人</strong>：用 <strong>@userinfobot</strong>，发送 <code>/start</code> 获取数字 ID</li>
<li><strong>群组</strong>：先把机器人拉进群，再用 <strong>@getidsbot</strong> 获取群 ID（通常以 <code>-</code> 开头，<strong>负号要保留</strong>）</li>
</ul>
<h3>3. 填入 RSSFlow</h3>
<ol>
<li>填写 Bot Token 与 Chat ID</li>
<li>勾选需要的推送类型，打开总开关</li>
</ol>
<h3>4. 测试与激活</h3>
<ol>
<li><strong>先</strong>打开与机器人的对话，点击 <strong>Start</strong> 或发送 <code>/start</code></li>
<li>再点 <strong>发送测试消息</strong></li>
</ol>
