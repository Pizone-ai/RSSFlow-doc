<h2>入口</h2>
<ul>

<li>暗色 / 浅色主题切换（默认暗色）</li>
</ul>
<h2>背景主题</h2>

<li><strong>暗色</strong>：星空背景与星系雾效</li>

</ul>

<table><thead><tr>
<th>视口宽度</th>
<th>布局</th>
</tr></thead><tbody>
<tr>

<td>单列；AI 区以底部抽屉展示</td>

<td>两列：AI 侧栏 + 正文</td>
</tr>
<tr>
<td>≥ 1280px</td>
<td>正文最大宽度约 900px</td>
</tr>
</tbody></table>

</ul>
<h2>AI 侧栏</h2>
<p>显示当前文章的：</p>
<ul>
<li>AI 摘要</li>
<li>要点列表</li>
<li>标签、评分、关键词</li>
<li>生成 / 刷新状态</li>
</ul>
<p>未配置 AI 时，触发摘要会提示前往 AI 设置。</p>
<h2>阅读进度</h2>
<p>顶部细进度条，随滚动更新。</p>
<h2>快捷键</h2>
<table><thead><tr>
<th>按键</th>
<th>动作</th>
</tr></thead><tbody>
<tr>
<td><code>J</code> / <code>→</code></td>
<td>下一篇</td>
</tr>
<tr>
<td><code>K</code> / <code>←</code></td>
<td>上一篇</td>
</tr>
<tr>
<td><code>Space</code></td>
<td>向下滚动约一屏（<code>Shift+Space</code> 向上）</td>
</tr>
<tr>
<td><code>F</code></td>
<td>切换收藏</td>
</tr>
<tr>
<td><code>R</code></td>
<td>标记已读并跳转下一篇</td>
</tr>
<tr>
<td><code>Enter</code> / <code>S</code></td>
<td>触发 AI 摘要</td>
</tr>
<tr>
<td><code>Esc</code></td>
<td>关闭阅读器</td>
</tr>
</tbody></table>
