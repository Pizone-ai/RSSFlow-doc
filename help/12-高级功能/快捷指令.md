# 快捷指令 / AI Commands

### 中文

内置与自定义提示词模板，对单篇或多篇文章一键做提炼、改写或专业研判。  
完整角色列表见 **AI 智能核心 → 专家指令集**。

## 入口

- AI 聊天 / 阅读侧栏 **指令中心**  
- 设置中的指令管理（创建 / 编辑）

## 自定义指令字段

| 字段 | 说明 |
|------|------|
| 名称 | 菜单显示名 |
| 提示词模板 | System 要求；可插入变量 |
| 上下文开关 | 是否附带摘要 / 对话历史 |

### 模板变量

- `{{summary_data}}` — 当前摘要与元数据  
- `{{original_context}}` — 原文  
- `{{history}}` — 链式任务中上一步输出  

### English

22 expert presets plus custom prompts. Full role list: **AI Smart Core → Expert Command Set**.

## Access

Command Center in chat/reader; manage under Settings.

## Custom Fields

Name, prompt template (`{{summary_data}}`, `{{original_context}}`, `{{history}}`), and toggles for summary/history context.
