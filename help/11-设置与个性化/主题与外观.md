# 主题与外观 / Theme

### 中文

在浅色、深色与跟随系统之间切换，覆盖侧边栏、设置、阅读器、聊天与图谱等界面。

## 切换

侧边栏工具栏 **主题** 图标（太阳 / 月亮）循环三种模式：

| 模式 | 适用 |
|------|------|
| 浅色 | 明亮环境 |
| 深色 | 夜间 / 暗光 |
| 跟随系统 | 与系统外观同步 |

选择保存在本地浏览器，重启后仍有效。

### English

Cycle Light / Dark / System via the sidebar theme control. Applies across sidebar, settings, reader, chat, and graph. Preference is stored locally.
