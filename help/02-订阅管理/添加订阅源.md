# 添加订阅源 / Adding Feeds

### 中文

## 入口

**设置 → 订阅源**

## 方法一：手动输入 URL

1. 在 URL 输入框粘贴 RSS 地址（示例：`https://www.engadget.com/rss`）
2. 点击 **添加订阅源**，或按 Enter
3. 成功后源会出现在「已订阅源」列表

## 方法二：导入 OPML / XML

1. 点击 **导入订阅文件**
2. 选择 `.opml` 或 `.xml` 备份文件（建议 UTF-8）
3. 导入结束后会显示摘要，例如：成功 X / 共 Y，失败 Z

## 方法三：从 SnagFlow 导入

推荐使用配套扩展 [SnagFlow](https://snagflow.oinchain.com) 把任意网页变成稳定 RSS，再通过 **CatchFlow** 一键导入。

详见帮助：**订阅源获取 (SnagFlow) → 配置 CatchFlow 导入**。

## 方法四：推荐订阅源

页面底部「推荐订阅源」按语言与分类展示，例如：

- **中文**：财经、科技数码、AI、区块链、商业、心理、综合资讯、二次元、软件、技术博客、产品设计、读书等
- **English**：Tech & Development、AI & Machine Learning、Global News、Business & Economics 等

点击条目旁的 `+` 即可添加；已订阅显示 `✓`。

### English

## Access

**Settings → Feeds**

## Method 1: Manual URL

1. Paste an RSS URL (e.g. `https://www.engadget.com/rss`)
2. Click **Add Feed** or press Enter
3. The feed appears in **Subscribed Feeds**

## Method 2: Import OPML / XML

1. Click **Import Feed File**
2. Choose a `.opml` or `.xml` backup (UTF-8 recommended)
3. A summary shows success / total / failed counts

## Method 3: Import from SnagFlow

Use the companion extension [SnagFlow](https://snagflow.oinchain.com) to turn almost any page into a stable RSS feed, then import via **CatchFlow**.

See **Feed Acquisition (SnagFlow) → Configure CatchFlow Import**.

## Method 4: Recommended Feeds

Browse categories at the bottom of the Feeds page. Click `+` to subscribe; subscribed items show `✓`.
