# 沉浸式阅读器 / Zen Reader

### 中文

沉浸式阅读器（Zen Reader）在新标签页提供全屏、无干扰的阅读体验，并集成 AI 摘要侧栏与键盘快捷操作。

## 入口

- 在侧边栏 **单击** 文章卡片
- 工具栏 **沉浸式阅读** 按钮

## 顶栏布局

- RSSFlow 水印（随明暗主题变化）
- 当前上下文（订阅源或分组名称）
- 打开原文链接
- 暗色 / 浅色主题切换（默认暗色）

## 背景主题

- **暗色**：星空背景与星系雾效
- **浅色**：暖白底色与轻量云层装饰

## 响应式布局

| 视口宽度 | 布局 |
|---------|------|
| &lt; 1000px | 单列；AI 区以底部抽屉展示 |
| ≥ 1000px | 两列：AI 侧栏 + 正文 |
| ≥ 1280px | 正文最大宽度约 900px |

## 正文排版

- 衬线字体，约 20px，行高 1.85，两端对齐
- 引用块、图片、代码与表格均做圆角与间距优化

## AI 侧栏

显示当前文章的：

- AI 摘要
- 要点列表
- 标签、评分、关键词
- 生成 / 刷新状态

未配置 AI 时，触发摘要会提示前往 AI 设置。

## 阅读进度

顶部细进度条，随滚动更新。

## 快捷键

| 按键 | 动作 |
|------|------|
| `J` / `→` | 下一篇 |
| `K` / `←` | 上一篇 |
| `Space` | 向下滚动约一屏（`Shift+Space` 向上） |
| `F` | 切换收藏 |
| `R` | 标记已读并跳转下一篇 |
| `Enter` / `S` | 触发 AI 摘要 |
| `Esc` | 关闭阅读器 |

在输入框内、或按住 Ctrl / Cmd / Alt 时，快捷键不会抢占浏览器默认行为。

> 完整说明见「阅读器核心操作 → Reader 快捷键大全」。

### English

Zen Reader opens in a new tab for full-screen, distraction-free reading with an AI sidebar and keyboard navigation.

## Access

- **Single-click** an article card in the sidebar
- Toolbar **Immersive Reader** button

## Top Bar

- RSSFlow watermark
- Context (feed or group name)
- Open original link
- Dark / light theme toggle (default: dark)

## Background Themes

- **Dark**: starfield and soft nebula accents
- **Light**: warm white base with light cloud accents

## Responsive Layout

| Viewport | Layout |
|----------|--------|
| &lt; 1000px | Single column; AI panel as bottom drawer |
| ≥ 1000px | Two columns: AI sidebar + content |
| ≥ 1280px | Content max-width ~900px |

## Typography

Serif body ~20px, line-height 1.85, justified. Quotes, images, code, and tables use rounded, readable spacing.

## AI Sidebar

Shows summary, bullet points, tags, score, keywords, and generation status. If AI is not configured, you are prompted to open AI Settings.

## Progress

A thin top progress bar tracks scroll position.

## Shortcuts

| Key | Action |
|-----|--------|
| `J` / `→` | Next article |
| `K` / `←` | Previous article |
| `Space` | Scroll down (~one screen; `Shift+Space` up) |
| `F` | Toggle favorite |
| `R` | Mark read + next |
| `Enter` / `S` | Generate AI summary |
| `Esc` | Close reader |

Shortcuts are disabled in text fields and when Ctrl / Cmd / Alt is held.

> See **Core Reader Operations → Keyboard Shortcuts** for details.
