# 添加订阅源

## 入口

设置 → 订阅源。

## 方法一：手动输入 URL

1. 在 URL 输入框中粘贴 RSS 订阅源地址
2. 占位符示例：`https://www.engadget.com/rss`
3. 点击"添加订阅源"按钮，或按 Enter 键提交
4. 系统会将该订阅源添加至您的列表中

## 方法二：导入订阅备份文件

1. 点击"导入订阅文件"按钮
2. 选择 `.opml` 或 `.xml` 格式的订阅备份文件
3. 系统将自动解析文件并批量添加订阅源
4. 导入完成后显示摘要：`成功导入 X / Y 个订阅源，Z 个失败`

## 方法三：从 SnagFlow 导入

参见 [03-订阅源获取-SnagFlow/配置CatchFlow导入.md](../03-订阅源获取-SnagFlow/配置CatchFlow导入.md)

推荐使用snagflow来生成订阅源，生成后可以一键加载进来，snagflow可以为任何文章页面生成稳定的RSS订阅源

## 方法四：推荐订阅源

页面底部提供"推荐订阅源"区域，按语言（中文/English）和分类展示：

- **中文分类**：财经、科技数码、AI、blockchain、商业、心理、综合资讯、二次元、软件、技术博客、产品设计、读书
- **English Categories**: Tech & Development, Tech News, AI & Machine Learning, Global News, Asia News, Art & Culture, Business & Economics, Philosophy & Rationality, Lifestyle

点击分类中的 `+` 按钮可直接添加（已订阅的源显示 `✓`）。

# Adding Feeds

## Access

Settings → Feeds tab.

## Method 1: Manual URL Entry

1. Paste an RSS feed URL into the input field
2. Placeholder example: `https://www.engadget.com/rss`
3. Click "Add Feed" button or press Enter
4. The system will process and add the feed to your list

## Method 2: Import Backup File

1. Click "Import Feed File" button
2. Select a `.opml` or `.xml` feed backup file
3. The system will automatically parse and bulk-add the feeds
4. Import summary displayed: `Imported X / Y feeds, Z failed`

## Method 3: Import from SnagFlow

See [03-SnagFlow/Configure-CatchFlow-Import.md](../03-订阅源获取-SnagFlow/配置CatchFlow导入.md)

We highly recommend using SnagFlow to generate feeds, which can then be imported with a single click. SnagFlow is capable of generating stable RSS feeds for almost any web page.

## Method 4: Recommended Feeds

The "Recommended Feeds" section at the bottom lists feeds by language (Chinese/English) and category:

- **Chinese categories**: Finance, Tech & Digital, AI, Blockchain, Business, Psychology, General News, Anime, Software, Tech Blogs, Product Design, Reading
- **English categories**: Tech & Development, Tech News, AI & Machine Learning, Global News, Asia News, Art & Culture, Business & Economics, Philosophy & Rationality, Lifestyle

Click the `+` button on any feed to add it directly (subscribed feeds show `✓`).
