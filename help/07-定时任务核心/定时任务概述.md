## 定时任务概述 / Workflow Overview

### 中文

定时任务功能允许您自动在后台抓取订阅源内容、运行 AI 进行分析，并将生成的情报或简报推送到您的手机端。

**入口：** 设置页面 → 侧边栏的“工作流”标签页。

**界面介绍：**
- **任务列表**：查看、启用、禁用、编辑或删除您创建的定时任务。您也可以点击运行按钮手动立刻执行任务。
- **执行历史**：查看任务的运行日志和历史记录，最多保留最近 50 条。

**核心说明：**
- **排队执行**：为了防止同时触发多个任务导致大模型 API 频率超限，系统会自动串行排队处理任务。如果一个任务正在进行 AI 分析，其他任务会在后台静默排队等待。
- **调度方式**：支持设定固定时间间隔（如每 12 小时执行一次）或定时触发。
- **推送渠道**：每个任务可以独立配置推送目标，支持 Telegram 机器人、飞书群机器人、浏览器桌面通知或自定义 Webhook。
- **使用限制**：免费用户最多同时启用 2 个定时任务，升级至专业版后可解锁更多额度。

### English

The Workflows feature allows RSSFlow Pro to automatically fetch feeds in the background, run AI analysis, and push intelligence reports to your preferred channels.

**Access:** Settings → "Workflow" tab in the sidebar.

**Features:**
- **Workflows**: View, enable/disable, edit, or delete scheduled tasks. You can also trigger any task manually with the run button.
- **Journal**: View task execution logs and history (stores up to the last 50 runs).

**Core Information:**
- **Sequential Execution**: To prevent concurrent tasks from hitting API rate limits, tasks are processed one by one in a serial queue. If a task is running, other triggered tasks will wait in the background queue.
- **Schedules**: Run tasks at fixed intervals (e.g., every 12 hours) or at specific times.
- **Push Channels**: Send reports to Telegram Bot, Feishu Bot, browser desktop notifications, or custom Webhooks.
- **Limits**: Free accounts can run up to 2 active workflows. Upgrade to Premium to unlock unlimited workflows.
