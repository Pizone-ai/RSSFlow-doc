# 专家指令集 / Expert Command Set

### 中文

专家指令是系统预置的**分析角色**。你在 AI 聊天里点选一条指令后，AI 会按该角色的关注点、写作结构和专业习惯，去处理**当前对话里已经加载的那些文章**（可以是某一天、某个标签组，或你手动选中的一批）。

和「高级功能 → 快捷指令」是同一套能力：

- **本页**：把 22 条内置指令**一条条讲清楚**（新手先看这里）。
- **快捷指令页**：讲入口在哪、怎么自定义、模板变量怎么写。

> 部分指令需要 **Pro** 才能用。你也可以自己加指令，会和系统 22 条出现在同一个菜单里。

## 怎么用（基础）

1. 先保证相关文章已经进了当前对话（例如：信息流里某分组的「AI 聊天」，或侧边栏打开 AI 聊天后再带入上下文）。
2. 在聊天输入区打开 **指令菜单 / 指令中心**。
3. **点一条指令** → AI 立刻用该角色处理当前文章，你一般**不用再手打长提示词**。
4. 等输出结束后，可以继续追问，例如：「把第三条展开」「用表格对比」。

**小建议：** 相关订阅源先打开 **自动 AI 摘要**，再跑专家指令，通常更稳、也更省费用。

## 一、全部 22 条指令介绍（请先读完这一节）

下面按编号列出全部内置指令。每条都写清：**它是干什么的、什么时候该点它、大概会得到什么**。

### 1. 创意构思者（Ideator）

从你当前加载的多篇文章里，找出「可以继续做内容」的话题方向，把零散新闻收成几个**话题簇**，并给出可写的角度。

- **什么时候用：** 不知道写什么、要做周选题、要给频道定主题。
- **你会得到：** 选题清单、话题分组、每个方向可以怎么切入的建议。
- **注意：** 它偏「找题」，不是直接写成终稿；成稿请再配草稿构建器。

### 2. 趋势筛选器（Trend Selector）

在已有资讯里判断哪些话题**更热、更值得优先跟**，按传播潜力或时效性做排序，并简单说明理由。

- **什么时候用：** 信息很多、只想抓今天该盯的 3～5 个热点。
- **你会得到：** 热点排序、为何重要、哪些可以先放一放。
- **注意：** 依据的是你订阅流里的内容，不是全网热搜榜。

### 3. 草稿构建器（Draft Builder）

把多篇素材整理成**一篇结构清楚的文章骨架或初稿**（标题、小标题、段落逻辑）。

- **什么时候用：** 素材已经够了，要开始写深度文、周报、综述。
- **你会得到：** 大纲，以及可继续改的正文草稿。
- **注意：** 初稿可能偏「正确但平」；对外发布前常用「人性化改写」润一遍。

### 4. 人性化改写（Humanizer）

专门改语气：减少「作为 AI」「首先其次最后」等机械感，让文字读起来像人写的。

- **什么时候用：** 已有摘要或草稿，但读着像机器；准备发公众号 / 邮件 / 对客文档。
- **你会得到：** 同一内容的自然表述版，结构通常保留，句子更顺。
- **注意：** 它主要改表达，不负责重新做事实核查（核查用事实简报）。

### 5. 风险雷达（Risk Radar）

从资讯里抓**风险信号**：地缘、监管合规、供应链、舆情、系统性冲击等，并尽量标出严重程度与触发条件。

- **什么时候用：** 做风控扫视、出海合规关注、不想漏「坏消息」。
- **你会得到：** 风险清单、可能影响谁、需要盯什么信号。
- **注意：** 是辅助扫描，不是法律意见或投资建议。

### 6. 行业雷达（Sector Radar）

按「赛道」看问题：供需、竞争格局、主要玩家、份额与产品/渠道壁垒。

- **什么时候用：** 研究某个行业、对比竞品、写行业一页纸。
- **你会得到：** 谁在场上、各自位置、机会与威胁。
- **注意：** 上下文里行业文章越集中，结论越靠谱。

### 7. 宏观视图（Macro View）

用宏观框架读新闻：增长、通胀、就业、流动性、央行与财政政策，以及它们对行业/资产的含义。

- **什么时候用：** 宏观周报、配置背景、解释「为什么市场在动」。
- **你会得到：** 主叙事、关键变量、对下游含义的提示。
- **注意：** 需要上下文里有宏观相关稿件；纯公司八卦稿效果一般。

### 8. 外汇量化（Forex Quant）

从利率、利差、风险偏好、宏观数据等角度，整理与汇率相关的逻辑（偏研究备忘，不是下单信号）。

- **什么时候用：** 读汇市、利率决议、跨境资金相关新闻。
- **你会得到：** 主要驱动因素、情绪方向、风险提示。
- **注意：** 不替代交易系统；勿当作「保证涨跌」结论。

### 9. 战略官视角（CSO Board）

站在公司战略层看：商业模式怎么赚钱、护城河在哪、价值如何被捕获、战略选项有哪些。

- **什么时候用：** 公司深度、财报季战略解读、竞品战略对比。
- **你会得到：** 模式拆解、壁垒判断、可选战略路径。
- **注意：** 更适合有公司/商业模式信息的文章集合。

### 10. 加密情报（Crypto Intel）

面向加密与 Web3：链上动态、DeFi 机制、代币经济、生态事件与常见风险点。

- **什么时候用：** Crypto 日报、协议研究、监管对加密的影响。
- **你会得到：** 事件要点、机制说明、风险点列表。
- **注意：** 波动与骗局多，输出需结合引文回看原文。

### 11. 事实简报（Fact Brief）

专门处理「说法互相打架」：把多源信息拆成**已较一致 / 仍矛盾 / 尚未证实**，减少情绪化转述。

- **什么时候用：** 争议事件、传闻与辟谣并存、需要冷静事实底稿。
- **你会得到：** 分栏式事实清单，而不是站队长文。
- **注意：** 以你订阅源为准；源本身偏差大时，结论也会受限。

### 12. 动态分析（Dynamics）

不满足于「发生了什么」，而追问**变量之间怎么互相影响**（因果链、正负反馈）。

- **什么时候用：** 政策—市场联动、行业连锁反应、复杂系统事件。
- **你会得到：** 因果说明、反馈环描述、关键节点。
- **注意：** 比摘要更「推理向」，上下文不足时会写清不确定处（质量依赖模型与素材）。

### 13. 推特长文（Threader）

把较长研究/综述压成适合 **X / Twitter** 的连续短帖（thread）：有钩子、有分条、有收尾。

- **什么时候用：** 研究结论要发社交、需要传播向改写。
- **你会得到：** 1/ n、2/ n… 结构的帖文草稿。
- **注意：** 为传播会简化表述，关键数字建议用引文再核对。

### 14. 市场简报（Market Brief）

整理成「可读的市场/行业简报」：上下游、逻辑链路、该盯的指标，结构接近投研备忘。

- **什么时候用：** 早报、周度市场备忘、给团队同步。
- **你会得到：** 分点简报、逻辑流、关注指标。
- **注意：** 和「行业雷达」比，更偏成稿结构；行业雷达更偏格局拆解。

### 15. 科技日报（Tech Daily）

聚焦科技与平台：大厂动向、云与 AI 基建、资本开支、平台壁垒与商业化进展。

- **什么时候用：** 科技线日报、大厂追踪、AI 基础设施相关阅读。
- **你会得到：** 要点列表 + 对商业/竞争含义的短评。
- **注意：** 适合科技订阅较多的上下文。

### 16. 去噪分析（De-noiser）

用尽量「第一性原理」的方式去掉废话、公关包装和正确的废话，抽出**核心命题、关键数字、可执行结论**。

- **什么时候用：** 论文腔、超长稿、PR 稿、信息密度低但必须读的材料。
- **你会得到：** 极简核心结论 + 关键参数。
- **注意：** 会大量删细节；需要细节时再对原文展开。

### 17. 技术博主（Tech Onboarding）

从工程实践角度评估新技术/工具/框架：学习成本、迁移摩擦、适用边界、演进风险。

- **什么时候用：** 技术选型、是否引入某开源组件、读技术公告。
- **你会得到：** 上手成本评估、迁移注意点、适不适合你场景的判断框架。
- **注意：** 上下文最好包含官方文档或深度技术文，而不仅是营销通稿。

### 18. 思维教练（Mind Coach）

针对「知道但做不到」：阅读拖延、决策瘫痪、信息过载，给出**低成本下一步动作**（微实验）。

- **什么时候用：** 信息太多无从下手、执行卡住、需要行动清单而不是再多分析。
- **你会得到：** 问题诊断 + 今天就能做的一小步。
- **注意：** 偏个人效能，不是心理诊疗。

### 19. 多米诺模拟（Domino Sim）

假设某个冲击已经发生，推演它在供应链/产业链上的**一阶、二阶、三阶**连锁影响。

- **什么时候用：** 断供、制裁、天灾、区域冲突外溢、关键节点停摆。
- **你会得到：** 分层影响清单（直接相关方 → 间接方 → 更远端）。
- **注意：** 是情景推演，不是预言；假设条件应在输出里写清。

### 20. 林迪过滤（Lindy Filter）

专门过滤「只有短热度」的噪音，保留更可能长期仍有价值的概念、原则与框架。

- **什么时候用：** 热点太多想沉淀知识、做长期笔记、去 FOMO。
- **你会得到：** 值得保留的原则列表，以及可丢弃的短噪。
- **注意：** 会主动「不追新」；若你要的就是热点运营，请用趋势筛选器。

### 21. Serenity 方法（Serenity Method）

审计系统里的**关键依赖与瓶颈**：物料/BOM、卡脖子环节、单点故障，以及可缓释方向。

- **什么时候用：** 制造、半导体、硬件、复杂供应链、关键零部件新闻。
- **你会得到：** 依赖清单、瓶颈节点、可能的备选/缓释思路。
- **注意：** 名称来自方法论风格的分析框架；重点在瓶颈，不在鸡汤。

### 22. 贝叶斯交易（Bayesian Trade）

用「情景 + 概率思维」讨论不确定事件：几种情形、各自大致含义、若判断错如何对冲（研究向）。

- **什么时候用：** 重大事件前后、路径高度不确定、需要决策备忘而非单点预测。
- **你会得到：** 情景表、粗略概率讨论、对冲/应对思路。
- **注意：** **不是**荐股或下单指令；概率为示意，取决于素材与模型。

## 二、进阶：怎么选、怎么组合（熟练后再看）

### 1）按目标快速挑选

| 你想做的事 | 优先点这些 |
|------------|------------|
| 找选题、抓热点、发社交 | 1 创意构思者 → 2 趋势筛选器 → 13 推特长文 |
| 写成文章并润色 | 3 草稿构建器 → 4 人性化改写 |
| 宏观 / 行业 / 公司战略 | 7 宏观视图、6 行业雷达、9 战略官、14 市场简报 |
| 风险与冲突信息 | 5 风险雷达、11 事实简报、19 多米诺、12 动态分析 |
| 科技与工程 | 15 科技日报、17 技术博主、16 去噪分析 |
| 加密与不确定决策 | 10 加密情报、22 贝叶斯交易 |
| 长周期与执行 | 20 林迪过滤、21 Serenity、18 思维教练 |

### 2）常用组合（流水线玩法）

同一批文章上，可以**按顺序点多条指令**（每条跑完再点下一条；需要时把上一段结论贴进追问里）：

| 组合名称 | 顺序 | 适合 |
|----------|------|------|
| 内容生产线 | 趋势筛选器 → 创意构思者 → 草稿构建器 → 人性化改写 →（可选）推特长文 | 运营 / 写作 |
| 争议事件线 | 事实简报 → 动态分析 → 风险雷达 | 舆情 / 研究 |
| 公司深度线 | 去噪分析 → 战略官视角 → 市场简报 | 个股 / 公司研究 |
| 冲击推演线 | 事实简报 → 多米诺模拟 → 贝叶斯交易 | 黑天鹅情景 |
| 赛道扫描线 | 行业雷达 → 科技日报或加密情报 → 风险雷达 | 赛道周报 |

若你在**定时任务**里配置了串行 / 并行工作流，也可用自定义指令把类似流水线固化（详见定时任务相关帮助）。

### 3）自定义与变量

- 设置里可增删自己的指令，与 22 条一起显示。  
- 模板变量、字段说明见：**高级功能 → 快捷指令**。  
- 写自定义时尽量写清：角色是谁、输入是什么、输出必须长什么样、不要写什么。

### 4）效果不好时检查这三件事

1. 当前对话里是不是**载入了相关文章**（空上下文只会空泛发挥）。  
2. 是否已有 **AI 摘要**（没有的话先让摘要跑完）。  
3. 指令是否和素材匹配（例如用「外汇量化」去跑纯娱乐八卦，不会有好结果）。

### English

Expert Commands are built-in **analysis roles**. Pick one in AI Chat and the model processes **articles already loaded in the conversation**.

- **This page:** full catalog of all 22 commands (start here).  
- **AI Commands page:** where to click, custom prompts, template variables.

> Some commands need **Pro**. Custom prompts appear in the same menu.

## Basics

1. Load relevant articles into the chat context.  
2. Open the command menu.  
3. Tap a command — no need to write a long prompt.  
4. Follow up in chat if needed.

Prefer **auto-summary** on feeds before heavy commands.

## Part 1 — All 22 commands (read this first)

### 1. Ideator
Finds expandable topic clusters and writing angles from your loaded articles.  
**Use when:** you need topics for the week. **Output:** topic map + angles. **Not:** a final polished essay (use Draft Builder next).

### 2. Trend Selector
Ranks which topics look hottest / highest priority in *your* feeds.  
**Use when:** too much news, need top 3–5. **Output:** ranked list + why. **Not:** a global social firehose ranking.

### 3. Draft Builder
Turns materials into an outline or first full draft.  
**Use when:** research is enough to write. **Output:** structure + draft body. **Tip:** run Humanizer before publishing.

### 4. Humanizer
Rewrites stiff/AI-sounding text into natural prose.  
**Use when:** draft exists but reads robotic. **Output:** same ideas, better voice. **Not:** fact-checking (use Fact Brief).

### 5. Risk Radar
Surfaces geopolitical, regulatory, supply-chain, and systemic risk signals.  
**Use when:** risk scan / compliance awareness. **Output:** risk list + triggers. **Not:** legal advice.

### 6. Sector Radar
Maps supply/demand, competitors, share, and moats for an industry.  
**Use when:** sector one-pagers. **Output:** landscape + threats/opportunities.

### 7. Macro View
Reads news through growth, inflation, liquidity, and policy.  
**Use when:** macro weekly context. **Output:** narrative + key variables.

### 8. Forex Quant
Organizes rates, differentials, and sentiment relevant to FX.  
**Use when:** FX / rates news. **Output:** drivers + caveats. **Not:** trade signals.

### 9. CSO Board
Strategy lens: business model, moat, value capture, options.  
**Use when:** company deep dives. **Output:** model breakdown + strategic options.

### 10. Crypto Intel
On-chain, DeFi, tokenomics, ecosystem events and risks.  
**Use when:** crypto daily / protocol research. **Output:** events + mechanisms + risks.

### 11. Fact Brief
Splits contested claims into agreed / disputed / unconfirmed.  
**Use when:** conflicting sources. **Output:** calm fact table, not a hot take.

### 12. Dynamics
Causal chains and feedback loops across variables.  
**Use when:** policy–market linkage or multi-step industry effects.

### 13. Threader
Compresses long analysis into X/Twitter-style threads.  
**Use when:** distribution. **Tip:** re-check numbers via citations.

### 14. Market Brief
Research-memo style market/industry briefing with logic flow and metrics to watch.

### 15. Tech Daily
Big-tech, platforms, CapEx, AI infra, commercialization notes.

### 16. De-noiser
Strips fluff/PR; keeps core claims and key numbers.  
**Use when:** long or low-density documents.

### 17. Tech Onboarding
Engineering view of adoption cost, migration friction, and fit.  
**Use when:** tooling decisions.

### 18. Mind Coach
Turns overwhelm into a tiny next action.  
**Use when:** execution stuck. **Not:** therapy.

### 19. Domino Sim
First / second / third-order shock propagation across chains.  
**Use when:** black-swan scenarios. **Label assumptions.**

### 20. Lindy Filter
Keeps longer-lived ideas; drops short-lived noise.  
**Use when:** knowledge retention (not trend chasing).

### 21. Serenity Method
BOM / bottleneck / choke-point audit for complex systems and hardware chains.

### 22. Bayesian Trade
Scenario + probability framing and hedge *ideas* under uncertainty.  
**Not:** stock picks or guaranteed odds.

## Part 2 — Advanced selection & combos

### Quick pick

| Goal | Try |
|------|-----|
| Topics / social | Ideator → Trend Selector → Threader |
| Write & polish | Draft Builder → Humanizer |
| Macro / sector / firm | Macro View, Sector Radar, CSO Board, Market Brief |
| Risk / conflict | Risk Radar, Fact Brief, Domino Sim, Dynamics |
| Tech / eng | Tech Daily, Tech Onboarding, De-noiser |
| Crypto / uncertainty | Crypto Intel, Bayesian Trade |
| Longevity / execution | Lindy Filter, Serenity, Mind Coach |

### Example pipelines

| Name | Order |
|------|--------|
| Content line | Trend Selector → Ideator → Draft Builder → Humanizer → (optional) Threader |
| Contested event | Fact Brief → Dynamics → Risk Radar |
| Company deep dive | De-noiser → CSO Board → Market Brief |
| Shock scenario | Fact Brief → Domino Sim → Bayesian Trade |
| Sector weekly | Sector Radar → Tech Daily or Crypto Intel → Risk Radar |

Run commands **one after another** on the same context; paste prior output into follow-ups if needed. Workflows can encode similar chains (see Scheduled Tasks docs).

### Custom prompts & troubleshooting

- Manage customs under Settings; variables on the **AI Commands** page.  
- Weak results? Check: loaded articles, auto-summary ready, command matches the material.
