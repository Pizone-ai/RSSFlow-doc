# 数据隐私说明 / Privacy & Data Security

### 中文

## 本地优先

默认数据保存在你的浏览器本地，不会自动上传到任意服务器。

## 存储位置

| 类型 | 位置 |
|------|------|
| 文章 / 摘要 / 评分等 | 浏览器本地数据库 |
| 设置、AI 配置、推送、工作流 | 本地存储 |
| 临时会话状态 | 内存；关闭浏览器后清除 |

## API 密钥

- 密钥保存在本地  
- 仅在你配置的供应商请求中使用，不发给第三方中转（除非你自己配置了中转 Host）  
- 密码框默认遮罩  

## 无追踪

不含分析 / 行为统计 / 遥测代码。

## 何时会离开本机

仅在你主动配置时发送必要数据：

- **AI**：文章内容 → 你配置的模型服务  
- **推送**：简报 → Telegram / 飞书等  
- **云门户 / MCP**：报告 / 桥接 → 你自己的 Worker / 站点  
- **激活**：许可证校验  

## 清理

删除订阅源会删除其历史文章；卸载扩展或清除站点数据会抹掉本地库，通常不可恢复。

### English

## Local-First

Data stays in the browser by default and is not auto-uploaded.

## Storage

Articles and AI fields in local DB; settings and workflows in local storage; session state in memory.

## API Keys

Stored locally, used only for your configured providers/hosts, masked in the UI.

## No Telemetry

No analytics or behavior tracking.

## When Data Leaves

Only integrations you enable: AI providers, push bots, your portal/MCP worker, license activation.

## Deletion

Deleting a feed removes its articles; uninstall / clear site data wipes local state.
