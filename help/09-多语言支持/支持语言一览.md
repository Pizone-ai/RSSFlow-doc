## 支持语言一览 / Supported Languages

### 中文

**14 种语言（源码  下拉框选项）：**

| 语言 | 代码 | 文件名 |
|------|------|--------|
| English | `en` |  |
| 简体中文 | `zh-cn` | `zh- |
| 繁體中文 | `zh-tw` | `zh- |
| 日本語 | `ja` |  |
| 한국어 | `ko` |  |
| Español | `es` |  |
| Português | `pt-br` |  |
| Français | `fr` |  |
| Deutsch | `de` |  |
| Italiano | `it` |  |
| Русский | `ru` |  |
| العربية | `ar` |  |
| हिन्दी | `hi` |  |
| Nederlands | `nl` |  |

**如何切换：**
Options → "常规"（General） → "语言" 下拉框

**切换后会即时生效：**
- 全部 UI 文本（侧边栏、选项页、聊天界面等）
- AI 生成的摘要和回复语言
- 聊天对话内容
- 自动化任务的报告语言
- 热点发现（Discovery）的标题和内容

**首次启动：**
- 自动检测浏览器语言并设置
- 保存在 浏览器本地存储 的设置中
- 手动选择后会覆盖自动检测

- 使用 React Context提供 `t()` 翻译函数
- 每个语言文件导出 key-value 映射
- 切换时调用 切换语言 并更新设置
- AI 调用时语言代码通过 `language` 参数传给 LLM
