# 沉浸式阅读器 / Zen Reader


---


---

## 全屏布局 Full-screen Layout

全屏沉浸式阅读体验。顶栏包含：

- RSSFlow 水印（呼吸动画，暗色/浅色模式自适应）
- 上下文指示器（如 "feed:XXX" 或上下文名称）
- 打开原文链接按钮
- 暗色/浅色主题切换按钮（localStorage ，默认 dark）

---

Full-screen immersive reading. Top bar includes:

- RSSFlow watermark (breathing animation, adapts to dark/light mode)
- Context indicator (e.g. "feed:XXX" or context name)
- Open original link button
- Dark/Light theme toggle (localStorage , default dark)

---

## 背景主题 Background Themes

- **暗色模式**：双层星空背景 `RealisticStarfieldBackground`（底层星光 + 顶层流星），紫色/蓝色星系核心雾效果
- **浅色模式**：蓝天云朵 `LightThemeBackground`（云层 + 飞机），暖白底色 `#fcfaf2`

---

- **Dark**: Layered starfield `RealisticStarfieldBackground` (bottom stars + top meteors), purple/blue galaxy nebula
- **Light**: Blue sky with clouds `LightThemeBackground` (clouds + airplane), warm white base `#fcfaf2`

## 响应式网格 Responsive Grid

| 视口宽度 | 布局 |
|---------|------|
| <1000px | 单列，AI 侧栏以底部抽屉形式出现 |
| ≥1000px | 两列网格：AI 侧栏 280-360px + 内容区 |
| ≥1280px | 内容区 max-width 900px，侧栏 300-360px |

移动端 AI 抽屉占 75vh，通过 spring 动画弹出。

---

| Viewport | Layout |
|---------|--------|
| <1000px | Single column, AI sidebar as bottom drawer |
| ≥1000px | Two-column grid: AI sidebar 280-360px + content |
| ≥1280px | Content max-width 900px, sidebar 300-360px |

Mobile AI drawer is 75vh, spring animation.

## 排版 Typography

- 字体：Charter / Georgia serif，20px
- 行高：1.85
- 两端对齐 (text-align: justify)
- 标题：900 字重，暗色下带文字阴影，浅色下品牌蓝辉光
- 引用块：2.5rem 内边距，2rem 圆角，左侧大引号装饰
- 图片：1.5rem 圆角，阴影，宽 100%
- 代码：JetBrains Mono 等宽字体，圆角 0.4rem
- 表格：全宽，圆角，交替行高亮

---

- Font: Charter / Georgia serif, 20px
- Line-height: 1.85
- Justified alignment
- Headings: 900 weight, dark mode has text shadow, light mode has brand blue glow
- Blockquotes: 2.5rem padding, 2rem border-radius, large opening quote decoration
- Images: 1.5rem border-radius, shadow, 100% width
- Code: JetBrains Mono monospace, 0.4rem border-radius
- Tables: full-width, border-radius, alternating row highlight

## 阅读进度条

顶部粘性导航栏内的细进度条（3.5px 高）。监听 `scroll` 事件（passive），通过 `requestAnimationFrame` 更新 `scaleX`。渐变颜色：teal → blue → purple。暗色模式有辉光阴影。

---

Thin progress bar (3.5px) inside the sticky top nav. Listens to `scroll` event (passive), updates via `requestAnimationFrame` using `scaleX`. Gradient: teal → blue → purple. Dark mode has glow shadow.

## 快捷键 Keyboard Shortcuts

| 按键 | 动作 | 备注 |
|------|------|------|
| J / → | 下一篇 (navigate next) | 100ms 节流，连读优化 |
| K / ← | 上一篇 (navigate prev) | 同上 |
| Space | 向下滚动 85vh | 留 15% 视线重叠，Shift+Space 向上 |
| F | 切换收藏 | 乐观更新，失败回滚 |
| R | 标记已读 + 跳至下一篇 | 视觉闪烁反馈，在后台自动保存至本地 |
| Enter / S | 触发 AI 摘要 | 30 秒超时保护 |
| Escape | 关闭阅读器 | `window.close()` |

修饰键（Ctrl/Meta/Alt）组合全部放行。输入框/文本框/可编辑元素中不触发快捷键。长按重复被拦截。

---

| Key | Action | Notes |
|-----|--------|-------|
| J / → | Next article | 100ms throttle, rapid-read optimized |
| K / ← | Prev article | Same |
| Space | Scroll down 85vh | 15% overlap, Shift+Space scroll up |
| F | Toggle favorite | Optimistic update, rollback on failure |
| R | Mark read + next | Flash feedback, saved to local storage in the background |
| Enter / S | Trigger AI summary | 30s timeout protection |
| Escape | Close reader | `window.close()` |

Modifier keys (Ctrl/Meta/Alt) are all passed through. Keys are blocked in input/textarea/contenteditable. Long-press repeats are filtered.
