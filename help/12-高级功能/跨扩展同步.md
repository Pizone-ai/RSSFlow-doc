# 跨扩展同步 / Cross-Extension Sync

### 中文

如果您安装了其他支持数据对接的浏览器扩展插件，可以使用“跨扩展同步”功能。它允许 RSSFlow Pro 在后台将分析处理好的文章、AI 自动生成的摘要、重点提炼和智能评分，自动实时地广播给您指定的其他扩展插件，实现工具链之间的无缝联动。

**配置入口：** 设置页面 → 侧边栏的“跨扩展同步”标签页。

**配置选项说明：**
- **启用同步**：同步功能总开关。
- **授权接收的扩展 ID (Target ID)**：出于隐私与安全考虑，只有您在此处明确授权并填入的扩展 ID，才能接收到 RSSFlow 传输的数据。如果您填入星号 `*`，代表允许与您电脑里的所有扩展建立基本连接。
- **自动推送**：开启后，每当 RSSFlow 在后台完成对新文章的 AI 摘要、要点分析和打分，都会第一时间以最快速度自动发送给授权接收的目标扩展，无需您手动操作。

**共享的数据内容包括：**
一旦对接成功，目标扩展可以实时获取以下由 AI 增益的精细化内容：
- 文章基础信息：标题、原文链接、发布日期、所属订阅源名称。
- AI 深度增益：AI 优化的更可读的重写标题、AI 自动摘要、文章核心要点列表、多视角观点深度分析。
- 标签与评估：自动分类标签、智能匹配的关键词、文章质量评估得分。

**分步配置步骤：**
1. **第一步**：打开您需要对接的目标扩展，复制其在 Chrome 或 Edge 扩展管理页面（`chrome://extensions`）中显示的 32 位唯一扩展 ID。
2. **第二步**：打开 RSSFlow 的“跨扩展同步”设置页面。
3. **第三步**：在输入框中粘贴该扩展 ID。
4. **第四步**：打开“启用同步”和“自动推送”开关，开始实时数据同步。

### English

The Cross-Extension Sync feature allows RSSFlow Pro to instantly stream processed feed items, AI summaries, bullet-point takeaways, and quality ratings to other authorized browser extensions on your machine, enabling seamless workflow automation.

**Access:** Settings page → "Sync" tab in the sidebar.

**Configuration Parameters:**
- **Enable Sync**: The master toggle to turn on data streaming.
- **Authorized Target Extension ID**: For security, only the specific extension ID you whitelist here will receive your reading data. You can enter an asterisk `*` to allow query requests from any local extension.
- **Auto Push**: When checked, RSSFlow will automatically push article metadata and AI analysis outputs to the target extension the instant they are generated by the background queue.

**Transmitted Article Insights:**
Once connected, the target extension will receive the following enriched dataset:
- Basic Metadata: Original Title, web link, publication date, and source feed name.
- AI Enhancements: Readability-optimized rewritten title, automatic summaries, key takeaways (bullet points), and multi-perspective opinion synthesis.
- Tags & Quality: Auto-generated categorization tags, matched keywords, and the calculated article quality score.

**Setup Instructions:**
1. **Step 1**: Find the 32-character ID of the extension you want to sync with on the browser extensions page (`chrome://extensions`).
2. **Step 2**: Open RSSFlow's "Sync" settings page.
3. **Step 3**: Paste the 32-character Extension ID into the Target ID field.
4. **Step 4**: Toggle on "Enable Sync" and "Auto Push" to activate real-time synchronization.
