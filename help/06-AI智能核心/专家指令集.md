## 自定义

可在设置中增删改个人指令；与系统指令合并显示在聊天指令菜单。

## 用法

AI 聊天输入区打开指令菜单 → 选择角色 → 对当前上下文一键分析。

### English

22 pre-tuned analysis roles (research, macro, risk, writing, crypto, etc.). Some require Pro. Custom prompts merge into the same chat command palette.

Use: open the command menu in AI Chat → pick a role → run on the loaded articles.
