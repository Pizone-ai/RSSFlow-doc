# 高级参数说明 / Generation Parameters

### 中文

在 AI 设置中展开 **显示生成参数**，可调节输出风格与长度。

## 参数

| 参数 | 范围 / 默认 | 说明 | 建议 |
|------|-------------|------|------|
| Temperature | 0–2，默认 0.7 | 随机性 / 创造力 | 摘要 0.3–0.5；对话约 0.7；脑暴 0.7–0.9 |
| Top P | 0–1，默认约 0.7 | 核采样 | 多数情况只调 Temperature 即可 |
| Max Tokens | 100–65536 | 单次输出上限 | Basic 常约 8192；Advanced 常约 20000；过小会截断 |

## 恢复默认

相关区域提供 **恢复默认**（标签范围与参数区可能各自独立），用于回到出厂预设。

### English

Open **Show Generation Parameters** under AI Settings.

## Parameters

| Param | Range / default | Role | Tips |
|-------|-----------------|------|------|
| Temperature | 0–2 / 0.7 | Creativity | 0.3–0.5 summaries; ~0.7 chat; higher for brainstorming |
| Top P | 0–1 / ~0.7 | Nucleus sampling | Prefer tuning Temperature only |
| Max Tokens | 100–65536 | Max output length | ~8192 Basic / ~20000 Advanced; too low truncates |

## Defaults

Use **Restore defaults** where available to reset factory presets.
