# 卡片阅读与多列自适应 / Card Layout & Multi-column

## 文章卡片结构 Article Card Structure

标准模式卡片 (`ArticleCard` 组件) 包含以下层级：

```
┌─────────────────────────────────────┐
│ 订阅源名称   ✦/•   3分钟前    [☆]   │  ← 首行
│                                     │
│ 标题文字 (serif, 19px, bold)        │  ← 标题
│                                     │
│ AI摘要文本 (13.5px, 3行限制)   [图] │  ← 主体区域
└─────────────────────────────────────┘
```

- **首行**：订阅源名称（截断）+ AI 指示器（✦ 蓝色=有摘要 / • 灰色=无摘要）+ 相对时间 + 书签按钮（hover 显示）
- **标题**：serif 字体，19px，font-bold，hover 变品牌蓝色
- **摘要**：`GPTsummary` 或 `contentSnippet`，13.5px，最多 3 行 (line-clamp-3)
- **缩略图**：可选（`imageUrl` 字段），64×64（640px 以上 80×80），hover 弹出大图预览

极简模式 (`MiniArticleCard`)：仅显示 AI 指示点 + 标题 + 时间，紧凑行布局。

---

Standard mode (`ArticleCard` component):

```
┌─────────────────────────────────────┐
│ Feed Name   ✦/•   3m ago    [☆]    │  ← Header row
│                                     │
│ Title (serif, 19px, bold)           │  ← Headline
│                                     │
│ AI Summary (13.5px, 3 lines)  [IMG]│  ← Body area
└─────────────────────────────────────┘
```

- **Header**: feed name (truncated) + AI indicator (✦ blue=has summary / • gray=no summary) + relative time + bookmark button (hover reveal)
- **Title**: serif, 19px, font-bold, turns brand-blue on hover
- **Summary**: `GPTsummary` or `contentSnippet`, 13.5px, max 3 lines (line-clamp-3)
- **Thumbnail**: optional (`imageUrl` field), 64×64 (80×80 at 640px+), hover for preview

Mini mode (`MiniArticleCard`): AI dot + title + time, compact row layout.

## 多列自适应 Multi-column Grid

列数计算：`Math.max(1, Math.floor(sidebarWidth / 340))`。当侧边栏宽度 ≥680px 时显示 2 列，≥1020px 时显示 3 列，以此类推。

- 使用 高性能虚拟列表 虚拟滚动库实现高性能列表
- 每列的 列列表渲染器 独立渲染
- 文章按日期分组，每组可跨列
- 拖拽调整大小时冻结列数（`isResizing` 为 true 时不重新计算），松手后结算

---

Column count: `Math.max(1, Math.floor(sidebarWidth / 340))`. Shows 2 columns at ≥680px, 3 at ≥1020px, etc.

- 采用虚拟列表技术，确保在展示成百上千篇文章时依然保持极佳的滚动流畅度
- Each column has an independent 列列表渲染器
- Articles are grouped by date, groups can span columns
- Column count is frozen during resize (`isResizing`), recalculated on release

## 标准 vs 极简模式 Standard vs Mini Mode

通过工具栏按钮切换，保存在 浏览器本地存储 的 `sidebar_view_mode`。

| 特性 | Standard | Mini |
|------|----------|------|
| 卡片高度 | 完整（含摘要和缩略图） | 紧凑（仅标题行） |
| 摘要 | 显示 3 行截断 | 不显示 |
| 缩略图 | 显示 | 不显示 |
| AI 指示 | ✦ 或 • 图标 | 彩色圆点 |
| 书签 | hover 显示图标 | 不显示 |
| 日期分组 | 完整样式 + 渐变装饰条 | 简约标签 + 品牌色左竖条 |

---

| Feature | Standard | Mini |
|---------|----------|------|
| Card height | Full (with summary + thumbnail) | Compact (title only) |
| Summary | 3-line truncated | Hidden |
| Thumbnail | Shown | Hidden |
| AI indicator | ✦ or • icon | Colored dot |
| Bookmark | Hover reveal | Hidden |
| Date groups | Full style + gradient bar | Simple label + brand left bar |
