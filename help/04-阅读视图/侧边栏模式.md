# 侧边栏模式 / Sidebar Mode

### 中文

日常主界面：浏览列表、整理未读 / 收藏，并跳转到图谱、信息流等全屏视图。

## 标签

| 标签 | 说明 |
|------|------|
| 未读 | 默认；显示未读角标 |
| 收藏 | 已收藏文章 |
| 图谱 | 新标签打开知识图谱 |
| 信息流 | 新标签打开 Flow |

## 文章卡片

- 展示来源、标题、摘要（优先 AI）、时间  
- 悬停可收藏；悬停 AI 指示可预览要点  

## 点击行为

| 操作 | 结果 |
|------|------|
| 单击 | 打开沉浸式阅读器 |
| 双击 | 标记已读 + 木鱼音效，卡片收起移除 |

## 悬浮操作

刷新、全部标记已读、新手指引。

## 工具栏

标准 / 极简、阅读器、图谱、AI 发现、AI 聊天、设置、主题。

### English

Main sidebar for scanning, favorites, and jumping to full-screen Graph / Flow.

## Tabs

Inbox, Favorites, Graph (new tab), Flow (new tab).

## Cards

Source, title, summary (AI preferred), time; hover favorite; hover AI indicator for bullets.

## Clicks

Single-click → Zen Reader; double-click → mark read with sound and collapse.

## FAB & Toolbar

Refresh, mark-all-read, help; plus view density, reader, graph, discovery, chat, settings, theme.
