# SnagFlow 概述

## 什么是 SnagFlow？

SnagFlow 是一个**本地优先**的浏览器扩展，能将任意网页转化为**实时 RSS 订阅源**。它是 RSSFlow Pro 的官方配套工具，通过 CatchFlow 集成实现无缝联动。

## 核心能力

- 将任意网页"视觉化抓取"（visual snagging）为 RSS 格式
- 在本地生成 RSS 订阅源，无需服务器
- 支持自动化更新

## 与 RSSFlow 的集成

SnagFlow 生成的可订阅规则可以通过 RSSFlow Pro 的设置页面中的 **CatchFlow**面板导入，无需手动输入 URL。

# SnagFlow Overview

## What is SnagFlow?

SnagFlow is a **local-first** browser extension that transforms any webpage into a **live RSS feed**. It is the official companion tool for RSSFlow Pro, integrating seamlessly via the CatchFlow panel.

## Core Capabilities

- "Visually snag" any webpage into RSS format
- Generate RSS feeds locally, no server required
- Support automated updates

## Links

| Resource | URL |
|----------|-----|
| Website | `https://snagflow.oinchain.com` |
| Chrome Web Store | `https://chromewebstore.google.com/detail/snagflow-rss-converter/beafcjcdnfeajjpmienpdjmeabhjnljh` |

## Integration with RSSFlow

SnagFlow-generated feed rules can be imported through the **CatchFlow** panel in RSSFlow Pro's settings page, eliminating the need for manual URL entry.
