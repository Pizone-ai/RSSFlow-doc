# 界面导览 / Interface Guide

### 中文

## 侧边栏布局

RSSFlow Pro 以浏览器 **侧边栏** 形式运行，包含四个标签页与底部工具栏。

## 四个标签页

| 标签 | 说明 |
|------|------|
| 未读 | 未读文章列表，支持无限滚动 |
| 收藏 | 已收藏的文章 |
| 图谱 | **在新标签页**打开知识图谱，便于多关键词组合搜索 |
| Flow 视图 | **在新标签页**按日期 / 标签查看 AI 处理后的信息流 |

> 点击「图谱」或「Flow 视图」时，会在新浏览器标签中打开，而不是在侧边栏内嵌显示。

## 底部工具栏

工具栏位于侧边栏底部（宽度较宽时可能显示在右侧）：

| 按钮 | 功能 |
|------|------|
| 视图模式 | 标准模式 ↔ 极简模式（极简可扫视更多条目） |
| 沉浸式阅读 | 在新标签打开阅读器。推荐快捷键：`R` 已读、`Space` 滚动、`F` 收藏 |
| 智能图谱搜索 | 打开图谱搜索 |
| AI 发现 | 基于 AI 处理后的数据评估趋势热点 |
| AI 聊天 | 进入对话；可切换顶部分组，或在左侧过滤数据 |
| 设置 | 打开选项页 |
| 主题 | 浅色 / 深色 / 跟随系统 |

## 选项页面

工具栏 **设置**，或右键扩展图标 → **选项**。共 9 个标签：

| 锚点 | 名称 |
|------|------|
| `#general` | 常规 |
| `#feeds` | 订阅源 |
| `#ai` | AI 设置 |
| `#notifications` | 消息推送 |
| `#sync` | 跨扩展同步 |
| `#automation` | 工作流 |
| `#mcp` | MCP 设置 |
| `#account` | 我的账户 |
| `#help` | 帮助 / 问题反馈 |

## 主题切换

选项页底部与侧边栏工具栏均提供主题切换：浅色、深色、跟随系统。

### English

## Sidebar Layout

RSSFlow Pro runs as a browser **sidebar** with four tabs and a bottom toolbar.

## Four Tabs

| Tab | Description |
|-----|-------------|
| Inbox | Unread articles with infinite scroll |
| Favorites | Bookmarked articles |
| Graph | Opens Knowledge Graph **in a new tab** for multi-keyword search |
| Flow View | Opens Flow **in a new tab**, grouped by date or tags |

> Graph and Flow open in a new browser tab, not inside the sidebar.

## Toolbar

| Button | Function |
|--------|----------|
| View mode | Standard ↔ Mini (scan more items) |
| Immersive Reader | Zen Reader in a new tab. Shortcuts: `R` read, `Space` scroll, `F` favorite |
| Graph Search | Open graph search |
| AI Discovery | Trend / hotspot analysis on AI-processed data |
| AI Chat | Chat context; switch groups or filter on the left |
| Settings | Options page |
| Theme | Light / Dark / System |

## Options Page

Toolbar **Settings**, or right-click the extension → **Options**. Nine tabs:

| Anchor | Name |
|--------|------|
| `#general` | General |
| `#feeds` | Feeds |
| `#ai` | AI Settings |
| `#notifications` | Notifications |
| `#sync` | Sync |
| `#automation` | Workflow |
| `#mcp` | MCP Settings |
| `#account` | Account |
| `#help` | Help / Feedback |

## Theme Toggle

Available in the options footer and sidebar toolbar: Light, Dark, System.
