# 云报告门户 / RSSFlow AI Report Portal

### 中文

定时任务生成的 AI 报告可同步到你的独立站点，便于手机阅读与分享。

## 特点

- 碎片资讯 → 结构化博文  
- 跨设备阅读，无需打开扩展  
- 可分享、利于归档  
- 分类 / 系列 / 标签归档  

演示：[blog.oinchain.com](https://blog.oinchain.com)

## 配置（任务报告设置）

| 字段 | 说明 |
|------|------|
| 站点 URL | 门户地址 |
| Auth Token | 发布鉴权 |
| 默认语言 | 中 / 英等 |
| 分类 / 系列 | 栏目映射 |
| 作者 / 标签 | 署名与标签 |

## 流程

1. 工作流生成报告  
2. 扩展安全推送到站点  
3. 站点发布为可分享页面  

相关：**定时任务 → 三种报告发布模式**。

### English

Publish workflow AI briefings to your personal portal for mobile reading and sharing.

## Features

Structured posts, cross-device access, shareable URLs, category/series/tag archives.  
Demo: [blog.oinchain.com](https://blog.oinchain.com)

## Config

Site URL, auth token, language, category/series, author/tags — under workflow report settings.

## Flow

Workflow generates → extension pushes securely → portal publishes a unique page.  
See **Report Publish Modes**.
