# 界面导览

## 侧边栏布局

RSSFlow Pro 以 Chrome 侧边栏（sidebar）的形式运行，包含四个标签页和底部工具栏。

## 四个标签页

| 标签 | 中文 | English | 说明 |
|------|------|---------|------|
| `all` | 未读 | Inbox | 显示未读文章列表，支持无限滚动 |
| `favorites` | 收藏 | Favorites | 已收藏的文章 |
| `graph` | 图谱 | Graph | **在新标签页**打开图谱页面，会有更充足的空间进行显示和发起搜索，支持多关键词组合搜索 |
| `flow` | Flow 视图 | Flow View | **在新标签页**打开flow视图，显示日或标签视图，可以看到每天AI处理过的数据|

> 点击"图谱"或"Flow 视图"标签时，会在新浏览器标签页中打开对应页面，而非在侧边栏内显示。

## 底部工具栏

工具栏位于侧边栏底部（宽度 ≥ 800px 时显示在右侧），包含以下按钮：

| 按钮 | 功能 |
|------|------|
| 视图模式切换 | 标准模式 ↔ 极简模式 极简模式可以查看到更多的内容 |
| 沉浸式阅读 | 在新标签页打开阅读器模式，进入到文章阅读页面，这是身为一个rss阅读器的核心，强烈推荐使用快捷键进行快速切换：R 标记已读、空格 滚动、F收藏 |
| 智能图谱搜索 | 打开图谱搜索页面 |
| AI 发现 | 打开 AI 热点发现页面，定时根据基础的AI处理后的数据进行评估当前趋势热点 |
| AI 聊天 | 打开 Flow 页面并进入当前分组聊天上下文 可以点击顶部分组进行切换分组数据，也可一个点击左侧进行数据搜索或过滤 |
| 设置 | 打开选项页面 |
| 主题切换 | 浅色/深色/跟随系统 |

## 选项页面

点击工具栏"设置"按钮或右键图标 → 选项，打开 选项页面。共有 9 个标签页：

| URL 锚点 | 中文 | English |
|----------|------|---------|
| `#general` | 常规 | General |
| `#feeds` | 订阅源 | Feeds |
| `#ai` | AI 设置 | AI Settings |
| `#notifications` | 消息推送设置 | Notifications |
| `#sync` | 跨扩展同步 | Sync |
| `#automation` | 工作流 | Workflow |
| `#mcp` | MCP 设置 | MCP Settings |
| `#account` | 我的账户 | Account |
| `#help` | 帮助/问题反馈 | Help/Feedback |

## 主题切换

选项页面底部和侧边栏工具栏均有主题切换按钮，支持三种模式：浅色、深色、跟随系统。

# Interface Guide

## Sidebar Layout

RSSFlow Pro runs as a Chrome sidebar with four tabs and a bottom toolbar.

## Four Tabs

| Tab | ID | Description |
|-----|----|-------------|
| Inbox | `all` | Unread articles, supports infinite scroll |
| Favorites | `favorites` | Bookmarked articles |
| Graph | `graph` | **Opens the Graph page in a new tab**, providing ample space to explore, visualize connections, and perform combined multi-keyword searches |
| Flow View | `flow` | **Opens the Flow View in a new tab**, displaying date or tag groupings where you can review daily AI-processed feeds |

> Clicking Graph or Flow View tabs opens the corresponding page in a new browser tab, not within the sidebar.

## Toolbar

The toolbar sits at the bottom (or on the right side when width ≥ 800px) with these buttons:

| Button | Function |
|--------|----------|
| View Mode Toggle | Standard ↔ Mini layout (Mini layout lets you fit and scan more content at once) |
| Immersive Reader | Opens Zen Reader in a new tab to access the article reading page. This is the heart of an RSS reader. We highly recommend using keyboard shortcuts for fast workflows: R to mark read, Space to scroll, and F to bookmark |
| Graph Search | Opens graph search page |
| AI Discovery | Opens the AI Discovery view, which evaluates and uncovers current trending hotspots based on historical AI-processed data |
| AI Chat | Opens the AI Chat window in Flow View. You can click top groups to switch conversation contexts, or search and filter data on the left panel |
| Settings | Opens options page |
| Theme Toggle | Light / Dark / System |

## Options Page

Click the Settings button in the toolbar or right-click the icon → Options to open 选项页面 with 9 tabs:

| URL Anchor | Tab Name |
|------------|----------|
| `#general` | General |
| `#feeds` | Feeds |
| `#ai` | AI Settings |
| `#notifications` | Notifications |
| `#sync` | Sync |
| `#automation` | Workflow |
| `#mcp` | MCP Settings |
| `#account` | Account |
| `#help` | Help/Feedback |

## Theme Toggle

Both the options page footer and the sidebar toolbar have a theme toggle button supporting three modes: Light, Dark, and System.
